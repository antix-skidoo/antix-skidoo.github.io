Hello. I put together this theme kinda by accident. Anti and I were discussing which icons to use in the control center and one thing led to another (the screenshot shows the icon set currently being used). Basically, the wall and window style are ones that I've been wanting to use for awhile. The gtk2 themes just sort of fell into place - I've been toying with the idea of having a light theme for users and a darker theme for root; I think it works quite nicely here. Conky is from erie.

I also took out a few things from the panel as these things are accessible from the keyboard, which I'm finding is much more convenient than mouse access (especially on a laptop).



Please enjoy.

The mixer icon is from the icon set in the screenshot. Just modify your ~/.icewm/toolbar file

Code: Select all

prog"Volume" /usr/share/icons/gTangish-2.0a1/16x16/apps/arts.png absvolume
and see my icewm thread in the tips and tricks section for the absvolume package.
