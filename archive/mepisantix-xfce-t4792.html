Some of you may know already.

MEPIS and antiX have come together to provide an xfce4.10 spin. This is still in alpha stage, but is already looking pretty good.

It is 'middleweight' (for you boxing fans) in that it is heavier than a typical antiX-full, but lighter than MEPIS kde. It seems to be pretty light as far as xfce distros go and includes more bells and whistles. It is based on Debian Wheezy and 'modernised' using dDebian-backports and MEPIS Community repos. Kernel used is 3.11.0.

It fits on a cd (just) and includes libreoffice.
grub2 is used
Includes antiX-installer (modified)
Includes antiX persistence and remastering of live set up
antiX live scripts used
antix-snapshot included
vlc for video
clementine for audio
live iso is multi-lingual
