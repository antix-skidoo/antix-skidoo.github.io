I've finally twigged that the issues I have had installing antix are down to legacy files in my existing /home folder. __{{emoticon}}__

I've had the same /home folder for a number of years on this lappy and installing antix led to so many problems, insane problems which should never had happened.

I tried deleting all the . dot files in /home once I got installed but during the user account set up issues had already been set in place.

Creating a clean user and then copying my data files over has allowed a stress free installation __{{emoticon}}__

If you have odd problems, especially with sessions and desktops then it may be worth trying it.
