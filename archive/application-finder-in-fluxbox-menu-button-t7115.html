hello anticapitalista and everyone  

menu button, start button, application menu button, start menu select application button,
in fluxbox there is no single button like this. fluxbox menu is found without any such button.  There is found from Jerry in 2008 created a fluxbox menu entry to AllApplications with summoning a file manager to show usr/share/applications.  Though it sometimes comes up that a whiskermenuesque button might be handy for some of us still halfway into fluxbox. From somewhere below the status bar found Here is a silly button way to have one such button,,, calling it more a remote control to take with you to set into a convenient place than calling it a start button; the fluxbox remote control for applications button; based on fluxbox menu and xfce4-appfinder.  
In fluxbox install repository package xfce4-appfinder.  If it is found that it is not able to edit the xfce4-appfinder entries check for installation of repository package exo-utils.  This is something like the old-fashioned practice to iconify a document that is our written favorites, links, etc.  The remote control button is a always on top link to a file; the file in this case is in the format of application finder.  The installed xfce4-appfinder will now be found in the fluxbox menu as Applications >[submenu] Accessories >[submenu] Application Finder.  There it is. the button example.  need it at the top first entry like that. A good example, But it is not okay to dedicate and lose the Accessories menu as the button, because we need things on that Accessories menu available.  The Application Finder entry would be good duplicated at the top slot of the Desktop > Other Desktops >[submenu] Application Finder [example Select from list] would be a good example to make more deliberate the switching of desktops would be okay, considering where this remote menu will be placed.  Or make a special menu item for this Select Menu button.  Here is made the detached fluxbox menu Select Menu addition with the Application Finder is the top entry and A Run Command entry second.  The Runrunner box is the execute command xfce4-appfinder --collapsed. (Alt + F2 is a different Runrunner box.) This menu, grab and tug with mouse,  can be jammed into the bottom left corner or anywhere you want.  With only a tiny bit showing of the second entry A Run Command should be accessible without being seen; Push it down a bit more to have Only the top entry Application Finder available will remain highlighted if it is the only one reachable.  A fluxbox menu hotline to applications and command center xfcestyle.
