3. Right click in a space in the folder (not on the file) - Window -> Terminal Here.

4. Type, sux press Enter, type your root password

5. Now type dpkg -i skype (use the Tab button to auto-complete)

6. If you get a message about depenency issues, type

apt-get -f install

7. If all is ok, update the icewm/fluxbox menu in Menu->Icewm/fluxbox Settings/Auto-update-menu.

8. Skype should appear in Menu->Applications->Internet.
