Welcome to antiX.

If you mean a full install to usb device like a full install to hard disk drive, then all is ok. Just make sure you install grub to the MBR of the usb device.
If you mean a persistent live install, then it can be done, but we are in the process of changing how we do this. The newer version will be better as you can have live persistence on one partition, in antiX-M11 and before you need 2 partitions.

If you use antiX-full, no need for 2 installs just to change window manager. In live mode, you get to choose at boot menu.

Hope this helps.
