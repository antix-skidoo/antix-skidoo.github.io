    The release ISO has shipped with the wrong asound.conf. This prevents the pre-amp being added to the alsamixer interface.
    The .desktop file is incorrect (should be hidden)
    The graphic file for the icon is the wrong one
    I'll resend everything.


    Because antiX saves the ALSA settings at shutdown it is neccessary to overwrite the saved settings file after the correct /etc/asound.conf is in place and the pre-amp activated. An easy way to set everything up is:
    • Put the correct asound.conf in place
    • Reboot to ensure the new configuration is read and used
    • Play any audio file (perhaps just the Test Sound in the Control Center)
      This places the Pre-Amp slider control in the alsamixer interface (usually at far right-hand side)
    At the next shutdown the new setting is stored and automatically used on each subsequent reboot.
