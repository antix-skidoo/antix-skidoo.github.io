On my machine, I have the full 13.2 version installed. I don't know anything about what languages are there, but on the antiX boot screen, if you press F2 you can select a language. IceWM and JWM are both there with SpaceFM as a filer option at the antiX login screen by pressing F1 repeatedly until it gets to the window manager and filer combination desired. I like the IceWM /SpaceFM combination best and the documentation says the languages are supported with Fluxbox and IceWM.

When I tried to mount the NFS partition by clicking it in spacefm, I got an error. But if I mount it using the root option, it does work. I think I would need to do this again anytime I rebooted the system.

If I wanted the NFS partition to mount automatically, I had to go edit the fstab file and on the 2 lines for that partition I changed the word"noauto" to"auto", saved the file, and then rebooted, and then it mounted automatically.

worktowork wrote:Please tell about fstab what I must write and where for mounting ntfs and vfat partition.
fstab
Example of vat:

Code: Select all

...
/dev/sda6 /media/DATEN vfat auto,users,gid=users,dmask=002,fmask=113,relatime 0 0
Example for nfs:

Code: Select all

...
/dev/sda5 /media/Musik ntfs uid=1000,umask=0022 0 0
The mount points DATA and MUSIC must be created before!

Code: Select all

mkdir /media/DATEN
