I had the same problem. I just did a clean install from CD of 8.2 over 8.0 (whiched worked perfectly) and found my Wicd would connect to the router as normal but the browser would not allow pages to load as if it they were blocked. After reading Anticapitalista's comment about the DHCP Client issue, I opened up the Wicd Network Manager program, clicked preferences menu then under the external programs tab changed the DHCP Client from automatic to dhclient. It then allowed web pages to load as normal in the brower.
