on the boot menu, you should be able to add the vga=773 parameter to the boot line. I believe you hit 'e' to edit an entry. Look for the word"quiet" and put your vga boot code there. he screen should tell you which key to hit to boot. Once you get a working resolution, you can use grub-customizer, (already included in MX) to permanently add the vga code.

fyi - the issue of boot codes not carrying over from livecd to installed system is one of the bugs being worked on for the next MX bugfix release, 14.3.
