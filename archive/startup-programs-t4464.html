It depends on the window manager you are using. Usually you need to edit a file in a hidden directory in your home directory. Usually it is $HOME/.$WINDOW_MANGER/startup. This is true for at least fluxbox, icewm, and jwm.

An easy way to get to these is via the Desktop tab of the antiX Control-Centre. There will be options to edit the settings of different window managers. You want to edited the"startup" file which will be one of the files available to edit.

First try running checkgmail from the command line. If you don't get a shell prompt in the terminal while it is running (you may need to hit <Enter> to get the prompt) then you need to put"&" after the command in the startup file. This launches the program in the background so other things can happen. If a"&" is needed but absent then only checkgmail will start and no other thing will launch on your desktop until you close (or kill) checkgmail.
