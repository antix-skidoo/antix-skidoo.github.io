Seems you edited the global configuration file â€“ which get's overriden by your personal configuration file. So edit the (hidden) file ~/.dillo/dillorc (short for /home/YOUR_USERNAME/.dillo/dillorc) instead. If it doesn't show up in your file manager toggle the view of hidden files by applying the key shortcut Ctrl h.

For customizations you generally should use the personal config files in your home directory, many of them in the hidden folder ~/.config.
