After installing hplip and the GUI interface, the hp-setup program had communications issues with the printer. I found out that in order to use the default mDNS/Bonjour network discovery method that the avahi-daemon had to be installed. Even then the GUI setup couldn't properly install the printer. It was necessary run"/usr/bin/hp-setup -i" interactive mode as root in order to get the printer to install correctly and show up in the HP Device Manager.
