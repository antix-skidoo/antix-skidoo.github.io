A tradução do Imagination não é ruim mas eu consertei umas falhas de digitação e outras coisinhas. Basta descompactar o arquivo imagination.mo.zip e colocar o arquivo resultante em /usr/share/locale/pt_BR/LC_MESSAGES/imagination.mo

O antiX veio sem a pasta de ajuda do Imagination. Se alguém reinstalou ele então essa pasta está presente mas ajuda é em inglês.
Para ter a ajuda e em português, descompacte o arquivo ajuda.zip e ponha a pasta html em /usr/share/doc/imagination/ (crie essa pasta antes se ela não existir). Note que a pasta dentro dela se chama en, mas isso não é um engano, é que o Imagination busca pela pasta com esse nome.
Pela linha de comando a operação é a seguinte: abra o terminal onde o ajuda.zip foi salvo (F4 no PCManFM), autentique-se como root (su) e:

Code: Select all

mkdir /usr/share/doc/imagination

Code: Select all

unzip ajuda.zip

Code: Select all

cp -r html/ usr/share/doc/imagination/html
Pronto! Agora ao chamar a ajuda de dentro do Imagination com a tecla F1 ou pelo menu de ajuda, ela estará em português.
