Supplemental to the description already provided in the announcement antix-16-released-t6434.html
here's a breakdown of"What additional stuff is provided in FULL version, compared to BASE version?"

iceWM (an alternative window manager), Synaptic (package manager GUI), easy networking/wireless setup,
scanner+webcam+bluetooth support, extra printer drivers, LibreOffice suite + dictionaries + extra fonts,
evince (PDF viewer/formfiller app), cherrytree (notekeeping app), mirage (imageviewer app), ufw (firewall GUI),
transmission (BitTorrent client app), hexchat (IRC client app), streamtuner (radio app), claws-mail (mailreader app),
gnome-mplayer (musicplayer GUI app), luckybackup app, extra game apps, openSSH server, python3.4, dosbox ...
