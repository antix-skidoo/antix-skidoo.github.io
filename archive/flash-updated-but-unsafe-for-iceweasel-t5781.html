close browsers (iceweasel/firefox, chromium, etc) for safe operation
AS ROOT:
nano / etc/apt/sources.list.d/mx.list
uncomment mepis repos
apt-get update
apt-get purge flashplugin-nonfree
apt-get -f install freshplayerplugin pepperflashplugin-nonfree
AS USER:
cp /usr/share/doc/freshplayerplugin/freshwrapper.conf.example ~/.config/freshwrapper.conf
nano ~/.config/freshwrapper.conf
change:

Code: Select all

pepperflash_path ="/opt/google/chrome/PepperFlash/libpepflashplayer.so"
to:

Code: Select all

pepperflash_path ="/usr/lib/pepperflashplugin-nonfree/libpepflashplayer.so"
change:

Code: Select all

enable_3d = 1
to:

Code: Select all

enable_3d = 0
change:

Code: Select all

enable_hwdec = 0
to:

Code: Select all

enable_hwdec = 1
Save and close.

Start iceweasel and navigate to :
about:plugins
verify new version of :Shockwave Flash
then try:

========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.adobe.com/software/flash/about/"
linktext was:"http://www.adobe.com/software/flash/about/"
====================================

and see if you get the flash header and number is what you expect from about:plugins
