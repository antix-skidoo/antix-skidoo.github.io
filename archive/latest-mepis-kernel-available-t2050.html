The latest 2.6.32-1-mepis-smp kernel is available in the repositories.

Make sure this repos is enabled in /etc/apt/sources.list

Code: Select all

deb ftp://ftp.mepis.com/mepis/ mepis-8.0 main
and then upgrade linux-image-2.6.32-1-mepis-smp and its headers via synaptic or

#apt-get update
#apt-get install linux-image-2.6.32-1-mepis-smp linux-headers-2.6.32-1-mepis-smp
#update-grub
