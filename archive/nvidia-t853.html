For direct driver install, you can use sgfxi, the smxi graphics driver script engine.

Type in: sgfxi -h
If something happens, it's installed. See the options.

If not, as root, do: cd /usr/local/bin; wget techpatterns.com/smxi.zip;unzip smxi.zip; sgfxi
