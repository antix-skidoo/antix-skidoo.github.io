With the increased complexity of Grub 2, I started to make a manager to help deal with all the manual file editing. However after starting to get back on it again to finish it I came across grub-customizer. It does everything my manager was going to do plus alot more. Do to how I found it slightly difficult to get a deb, when I finally found one I added it to the main section of the antix repo for anyone else looking at getting it to help out with grub 2. You can install it now by apt-get update && apt-get install grub-customizer.
