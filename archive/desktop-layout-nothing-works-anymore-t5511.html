    herbstluftwm is a tiling wm that does not use the mouse very much

    Try, pressing windows key Enter. This should open a terminal.
    In that terminal type desktop-session-exit - press enter
    The logout app should appear. Logout.
    At login screen use F1 key to toggle to the desktop you want.
