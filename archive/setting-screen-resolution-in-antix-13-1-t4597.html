@sessal - you should really start your own post, but you can find the setting in the antix control center on the"Session" tab,"Set Screen Resolution". If you need to make a persistent change, you may need to add"xandr -s 800x600" to your startup files. replace 800x600 with your desired resolution. If you have any more questions, please open your own post. Thanks.
My 2 cents. Go to antixcc (antix control center, crossed wrenches in icewm toolbar)

Pick edit Icewm. Go to startup. Put in startup file

Code: Select all

xrandr -s 1024x768
save it. Reboot. I run my Motorcycle Shop Desktop using this cheat because I have a 22 inch Dell CRT that
AntiX wishes to run the max resolution the crt can give and I need a magnifying glass to read the screen.
