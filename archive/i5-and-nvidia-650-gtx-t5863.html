My experience so far with Nvidia is to boot into run level 3 by appending the number 3 to the grub boot line. Login as root. Run sgfxi and follow the prompts. Change to run level 5. It may require a reboot to disable the nouveau video driver before it can actually install the Nvidia driver. The procedure is the same the second time around.
