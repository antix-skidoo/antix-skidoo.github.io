Nothing wrong, just that some apps do not get automatically added to the menu. You will have to do it manually.

For icewm menu.

Control Centre-Edit icewm Settings-menu

Add this line somewhere: prog"Skype" - skype

For fluxbox menu.

Control Centre-Edit fluxbox Settings-menu

Add this line somewhere: [exec] (Skype) {skype}

Don't know about the DC++ app as I though it only ran on windows.
