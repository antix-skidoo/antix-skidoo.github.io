I added"ipv6.disable=1" to my menu.1st, and it works well. I still have the option to enable ipv6 if needed.

Code: Select all

title  antiX at sda1, kernel 3.7.10-antix.4-amd64-smp
root   (hd0,0)
kernel /boot/vmlinuz-3.7.10-antix.4-amd64-smp root=/dev/sda1 quiet nomodeset modeset=0  ipv6.disable=1
initrd /boot/initrd.img-3.7.10-antix.4-amd64-smp
boot
