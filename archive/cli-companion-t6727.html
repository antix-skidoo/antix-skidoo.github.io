I added it to my personal menu after the install. Used edit menus in antixcc to add it to personal. Line entry for personal menu below.

Code: Select all

prog"CLI Companion" /usr/share/pixmaps/clicompanion.16.png clicompanion
You can add the above line to toolbar in icewm if you wish it. What it looks like on my install.
