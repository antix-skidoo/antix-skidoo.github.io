I find this very handy, it saves me having to remember where things are located in the menu.
  • Click Menu --> Settings --> Config - User --> Edit Fluxbox Menu
  • Add a line to the bottom of the first section so it reads:

    Code: Select all

             [exec] (Terminal) {wterm -tr +sb -fg white -T"antiX"}
         [exec] (Browser) {opera}
         [exec] (Email) {claws-mail}
         [exec] (Files) {rox}
         [exec] (All applications) {rox"/usr/share/applications"}
  • Clicking this entry will open a screen of all installed applications listed alphabetically
[/size]

Edit: just realized I have Opera as the command with browser--sorry, that was not the change I intended to highlight (although for me Opera is a perfect match with antiX).
