Amule it's not avalible for Jessie users.
Here's how you can get amule:

First, edit /etc/apt/sources.d/debian.list
copypaste the"jessie" line to a new line, and change"...jessie..." to wheezy
save the edited file then
sudo apt update && sudo apt install amule

Answer"yes" when prompted to install dependencies (wxwidgets v2.8 libraries) along with amule. About 22Mb total will be installed.

After install, again edit /etc/apt/sources.d/debian.list and remove (or outcomment, by placing a # symbol at beginning of line) the"wheezy" line.
(I would also then) again perform sudo apt update

ps: I've chosen to stick with amule version 2.3.1 (from wheezy repo).
If you pull from stretch repo, you'll get a later version of amule which uses wxwidgets v3.0 (and I haven't tested that).
So, heads up ~~ when you launch amule and it mentions"a newer version is available", yeah... thanks but no.
