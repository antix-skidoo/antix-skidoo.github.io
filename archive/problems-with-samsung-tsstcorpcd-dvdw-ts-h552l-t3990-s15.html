For a movie dvd, you do not mount it.

Go to gnome-mplayer and open it there.

Or type in a terminal as user not root once the dvd is inserted

mplayer /dev/sr0

You might also need to install libdvdcss2 from the deb-mutimedia repo. Enable it, then install libdvdcss2 and then disable multimedia repo.
