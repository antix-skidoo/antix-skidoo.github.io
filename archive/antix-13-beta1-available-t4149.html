If replacement pdf readers are being considered, Foxit-Reader is worth looking at. I have used it for a long time without a hitch. It is a good choice for both older and newer kit.
Main benefits:
  • It is tiny. Size after installation=6326kB
  • Has no dependecies
  • Requires very little RAM
  • Makes few demands on CPU
  • Is very responsive in use
  • How to use it is obvious and intuitive because its interface is familiar and simple
