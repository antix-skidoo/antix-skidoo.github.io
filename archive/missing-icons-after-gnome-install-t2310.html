Open Synaptic (System > Administration > Synaptic Package Manager) and search for 'icon'. You may find several icon themes like tango, gnome extra, gilouche, gnome colors etc.
Also go to System > Preferences > Appearance. In the 'Theme' tab, at the bottom, there's a link to obtain more themes online and it will open the browser in the Gnome Art page (

).

- Edit: if you download icons from Gnome Art's page (and there are several beautiful sets) all you have to do to install most of them is extract them to ~/.icons or /usr/share/icons (some have different ways but then there's a README file explaining that).
Also, depending on the icon set you're using, you can open its folder in /usr/share/icons with ROX-Filer as root (gksu rox /usr/share/icons/youriconset) and if theres no 'actions' folder then you right click on the 'stock' folder, choose 'Link...' and create a relative link for it called 'actions'.

I forgot to say: the Gnome icon sets are enough for the Gnome applications but some of antiX's apps (like PCManFM, XArchiver etc.) may have this issue of missing icons. Creating an 'actions' folder like I said above solves PCManFM.
There are maybe more generic icon sets for GTK in gnome-look.org. The ones that come with antiX (SimplyGrey and Buuf-Deuce) should fulfill the needs of antiX's apps (except Imagination that has some missing icons). Buuf-Deuce seems to be very complete.
