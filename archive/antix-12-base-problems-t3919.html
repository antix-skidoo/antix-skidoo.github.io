1. you need to edit the .desktop file of the menu (usr/share/applications/synaptic.desktop) and add before the synaptic command gksu (Exec=gksu synaptic) or use the antix control center. (i'm not sure if the base antixcc has synaptic, Manage Packages.)
2. the alternative window managers are not installed, you need to install them. the entries are there so you don't have to edit slim configuration to list them. also base in kind of spartan and only comes with a handful of applications that permit you to build from there. if you want more choices use full or install applications to base.

good luck and have fun with antix.
