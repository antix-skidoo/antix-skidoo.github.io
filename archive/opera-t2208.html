These steps work for any external browser you download and extract and put in /opt folder. Making the launcher is as easy as pie in any desktop except for maybe enlightment. Just a helpful hint. In /home in file manager I create a new folder called Icons(I know there are hidden folder in /home with icons but I do this anyways). Then I download from google images whatever icon I wish to use for launcher and place it in /home/Icons folder. Launcher script will always begin with /opt if you put it in that folder. That is just the way I do it. Doesn't mean you have to do it this way. Linux is about choice anyhows. This is just what works for me. Good luck with it. Happy Trails, Rok
