alfanhui, here it is.

1. Download these 3 files from
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.mepisimo.com/antix/Translation/Spanish/"
linktext was:"http://www.mepisimo.com/antix/Translation/Spanish/"
====================================
" onclick="window.open(this.href);return false

a) menu-es-fluxbox
b) menu-es-icewm
c) toolbar-es-icewm

2. Move a) to (hidden) .fluxbox folder in your /home directory and rename menu

Move b) to (hidden) .icewm folder in your /home directory and rename menu

Move c) to (hidden) .icewm folder in your /home directory and rename toolbar

3. (If you haven't already set up Spanish locales)
As root in a terminal
dpkg-reconfigure locales (let it finish) then
echo"export LANG=es_ES.UTF-8" >>/etc/profile
update-menus (as root and user in both icewm/fluxbox. You may need to logout first and do it again)

Edit /etc/X11/xorg.conf file so it reads (about line 50)

Code: Select all

  Option"XkbLayout""us,es"
  Option"XKbOptions""grp:alt_shift_toggle,grp_led:scroll"
EndSection
Logout/login
Now you can switch keyboard language from English to Spanish by using Alt-Shift
