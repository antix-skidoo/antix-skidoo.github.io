There are no archive extractors configured.
Attached are 12 common ones to be placed in /root/.config/rox.sourceforge.net/MIME-types/

Suggestion is these are included within the ISO.



The current Root Terminal entry defaults to running urxvt. This opens a very small window with extremely tiny text and dim text with no obvious way to change the terminal defaults. It is uncomfortable and difficult to use like this. For casual command line users it is not user friendly. It would be preferable to open ROXTerm as root. Changing its defaults is both obvious and easy.

Suggestion is that ROXTerm replaces urxvt as the default terminal opened by ROX-Filer as root.

To configure ROX-Filer (root)
Options-->Menu-->Terminal emulator program=roxterm

This leads to a more comfortable, consistent experience for casual command line users.
