ROX-Filer Icon Missing
prog"Rox Filer" roxfiler.png rox

This icon is not present in the system.
How about creating a .desktop file using a copy of the ROX icon in /usr/share/rox/.DirIcon. It might appear in Applications-->System Tools in a similar manner to MC and Space FM.

Example
Copy and convert /usr/share/rox/.DirIcon to /usr/share/icons/antiX/rox-filer.png
Create /user/share/applications/antix/rox-filer.desktop

Code: Select all

[Desktop Entry]
Name=ROX Filer
Comment=File Manager
Exec=rox-filer
Icon=/usr/share/icons/antiX/rox-filer.png
Type=Application
Categories=Application;System;
