Both Master Mono and Surround are muted. If you toggle them it should work fine:

amixer -c 0 sset 'Master Mono' toggle
amixer -c 0 sset 'Master Surround' toggle

If you want to change the off to on just repeat the same commands using the name of the device.

For changing the volume you should use:

amixer -c 0 sset 'Master Mono' 100%
amixer -c 0 sset 'Master Surround' 100%
P'têtre que simplement ta carte est"muted" (muette).
amixer fournit en ligne de commande les actions accessibles dans alsamixer:

on peut naviguer de canal en canal (les barres verticales) avec les flèches gauche droite du clavier.
Si a la base de la barre tu as [MM] au lieu de [00] -- dans le ptit carré vert -- le canal est muet.
On bascule de MM à 00 et inversement avec la touche m du clavier
la touche F6 permet de choisir la carte son, de voir quelle(s) carte(s) son est/sont accessible(s)
