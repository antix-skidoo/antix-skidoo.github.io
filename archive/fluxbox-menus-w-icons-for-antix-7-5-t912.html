I spent some time creating a fluxbox menu with icons, for Antix 7.5 . Just for anyone that wants it. From"Admin tools" to"WM's" the applications don't have icons because I just didn't know what to put there. Almost everything would look like a terminal. If someone wants to improve on that section, go for it. It of course uses icons that are already on Antix 7.5

The menu file has not been modified in any other way. Just icons added.

@OU812, Anti or anyone: I found it convenient to launch the entire control center application as root (using su wterm like other apps) so I didn't have to keep re-entering my password.
