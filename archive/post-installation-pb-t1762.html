Pierre:

The Fluxbox menu (or Icewm menu) can be configured to display and start applications you add. In full Antix, you can edit the menu by clicking Settings>Control Center>Edit Fluxbox Settings and then click on Menu. The syntax is pretty simple, for example

exec (abiword) {abiword}

Usually, apps you add will also show up under Full Fluxbox in the fluxbox menu (at least in the full antix version).

If for some reason you can't do this from control center (or maybe there's no control center in antix base; not sure about this), you can always edit the menu file directly

leafpad .fluxbox/menu

I hope this helps.

Pedro
