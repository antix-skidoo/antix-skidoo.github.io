Not really because the changes are in antiX's apps ao configs which aren't in any repo yet (are made locally).
But in this case the changes are slight and may not affect you.
If you have an Intel (on-board) graphic card then you may add the package libgl1-mesa-dri by, as root

Code: Select all

apt-get install libgl1-mesa-dri
and add

Code: Select all

Option"AccelMethod""UXA"
to /etc/X11/xorg.conf under the line `Driver"intel"'. See:
