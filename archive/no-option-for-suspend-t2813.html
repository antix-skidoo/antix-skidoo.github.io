With 8.5 have not found many wireless problems as long as you stay away from broadcom chips. I am not saying that they will not work or not work well, it is just not an"out of the box" configuration in my opinion. My familly has 4 laptops on wireless, one desktop wirless, and I have put antiX on another desktop and laptop both running wireless and they all work fine,"out of the box".

I think the best test for out of the box would be to try to run a live cd / live usb and see if it works. PS. if you would like to use WICD you would need to change the preferences, so I would suggest to use ceni for seeing if it would work.
