(iceWMCP) is an app that hasn't been updated for some years now but it's very complete and can be used as a control panel for system tasks too.
I wouldn't advise to use it in antiX full since the latter has it's own CP, has some tools of its own for Icewm too and has Icewm already setup. Since iceWMCP has a lot of options to tweak the window manager it would mess antiX's setup so it's advised to backup the files in ~/.icewm before playing with it.
I myself haven't tried it much but it seems quite evident that it's a nice tool if one installs Icewm in an antiX-base, for example.

I found a program that could be called the solution for auto-generated menus in IceWM. It's called MenuMaker:

========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://sourceforge.net/projects/menumaker/"
linktext was:"http://sourceforge.net/projects/menumaker/"
====================================

url was:"http://menumaker.sourceforge.net/"
linktext was:"http://menumaker.sourceforge.net/"
====================================

It simply reads Debian's menu files, .desktop files and other legacy types and creates a very neat and comprehensive IceWM menu (works with other window managers too).

I've installed it but I think it can be run from the decompressed folder. The basic run command is

Code: Select all

mmaker icewm
Besides that it has some option like to set a default terminal emulator different than Xterm etc.

To update the menu

Code: Select all

mmaker icewm -f
so it overwrites the menu file.

Some small fixes for IceWM themes that come with antiX:

1) AntiX-Green:

Code: Select all

# cd /usr/share/icewm/themes/AntiX-Green/

Code: Select all

# rm dframeARR.xpm

Code: Select all

# ln -s frameAR.xpm dframeAR.xpm
2) FauxGlass: the default.theme file in /usr/share/icewm/themes/FauxGlass/ has a mistake in the ColorNetIdle= line and also a bit confusing comment about something that wasn't working. I suggest leaving

Code: Select all

ColorCPUStatusIdle=""
and

Code: Select all

ColorNetIdle=""
I also suggest

Code: Select all

Look=pixmap
since it works very well with this theme. But some may prefer the flat look.
