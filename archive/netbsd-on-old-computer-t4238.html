    I replaced my old computer with a new one, on witch I installed antiX. About the old one I thought: let's give NetBSD a try. So now I am running NetBSD on this 1500+ amd/athlon computer. It's an surprisingly smooth experience. It took about 15 minutes to install. Next I installed xfce4 using pkgin, a binary-package manager, comparable to apt-get. The repo is quite limited and the packages are not really 'bleeding edge'. But that is just what this old machine needs. For instance I use an old rebranded Firefox 3.6 that is available besides Firefox 19.0. My installation weighs in for 2,0 GB totally. So you see: NetBSD doesn't require much diskspace.
