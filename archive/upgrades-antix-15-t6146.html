There will soon appear in the repos quite a few upgrades of antiX apps/libs used on antiX-15.
The vast majority are bug fixes and only a few will be 'enhancements' since antiX-15 was released.

You may/should get prompted to answer yes or no to keep your set up or accept the maintainers version.
There should be no problems answering either way if you have basically kept to the default repos.

eudev, at some point, will replace udev (though we packaged it as udev as a way to avoid upgrade issues).
From tests, there are 2 possible issues, One very minor IMO and the other might be more a 'deal-breaker'

1. new eudev/udev no longer uses eth or wlan for networking. You might need to use ceni/wicd to reconfigure your network

2. I have seen this on 64 bit version only - skype will not install. However, if it had already been installed, it should still work,
