Did you hit F1 in Imagination and no help page showed up?
The solution (as root)

Code: Select all

apt-get install --reinstall imagination-common
This brings back the (tiny) help page which was left out of antiX, probaby by mistake, in anti's effort to shrink the ISO file.

By the way, it apparently needs this also to import some audio files

Code: Select all

apt-get install libsox-fmt-all
