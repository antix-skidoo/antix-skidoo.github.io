1. One is for user, the other is for root.

2. No need to save elsewhere if using persistence and you make the savings in the user specified at login. They are saved in the user folder in /home.

3. Yes.

4. If you use rootfs=XXX then you can install apps and they will be kept on next reboot. You can then, if you wish, install to hard drive and all chages and new apps should carry over.

5. Unless there are drastic changes (very unlikely) you would simply replace the .iso file for the newer one, and if there is a new kernel the initrd.gz and vmlinuz files as well.
