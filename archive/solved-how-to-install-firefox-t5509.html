Unzip it in /home/boney.

Go into the folder firefox and left click firefox to make sure it opens.

Then make a Icewm menu or toolbar launcher with the path

/home/boney/firefox/firefox

or drag and drop firefox launcher in folder firefox if using rox pinboard to your desktop and pick symlink relative. Then right click it and look for set icon. Left click that.
Then look in /home/boney/firefox folder and poke around till you find firefox icon. Drag and drop that into the set icon window.

Happy trails, rok.

Edit: you will be updating firefox through firefox browser under the help section on firefox toolbar.
