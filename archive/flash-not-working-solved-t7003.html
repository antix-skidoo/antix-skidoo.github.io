I download the tar.gz from adobe's site and extract the latest libflashplayer.so and install it manually using drag and drop in rox file manager.

Code: Select all

harry@biker:~
$ locate libflashplayer.so
/etc/alternatives/libflashplayer.so
/home/harry/backup/Backups/libflashplayer.so
/usr/lib/flashplugin-nonfree/libflashplayer.so
/usr/lib/mozilla/plugins/libflashplayer.so
My /etc/alternatives and /usr/lib/mozilla /plugins are symlinks <relative> pointing to /usr/lib/flashplugin-nonfree . Firefox is closed when I do this.
I go this route because I use Seamonkey. That way when I update falsh manually. I only have to replace the old /usr/lib/flashplayer-nonfree/libflashplayerr.so . I can leave the symlinks alone.
