Found an IBM Thinkpad T41 laptop, from a store that refurbishes/resells stuff, $65.00. Came with a fresh install of WinXP Pro plus lots of opensource software. So partitioned and installed antiX for a dual boot ('tho I'm not using the Win part). My main use is taking it to the public library or cafe for the free wireless.

Ya, it's not antix-13-beta1, but this antiX version is apt-get update/dist-upgrade to current date, and working well.
512MB is more than enough, even with wicd, Iceweasel with 4 tabs open, plus say LOwriter, roxterm and more being used, ram usage hasn't gone beyond 50%.

So far the newer kernels all work, ie 3.7.1-antix.2-486-smp, 3.7.1-antix.3-486-smp, 3.7.10-antix.1-486, and kernel 3.8.0-antix.1-486.
Including going directly to the slim menu with this ATI grapics card. Kernel 3.5.0-antix.2-486-smp does not do this, but dumps you into tty1 needing a manual startx or slim command to take you to the gui.
