OK, after about 12 attempts, I finally got Dave server available... What I did again in latest successful attempt:
1. Purged antix-archive-keyring

Code: Select all

apt-get purge antix-archive-keyring
2. Removed package lists

Code: Select all

rm -rf /var/lib/apt/lists/*
3. Downloaded manually package
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://antix.daveserver.info/jessie/pool/main/a/antix-archive-keyring/"
linktext was:"http://antix.daveserver.info/jessie/poo ... e-keyring/"
====================================
and installed it using sudo gdebi
4. Updated package lists

Code: Select all

apt-get update
Then it worked FINALLY!!!
One note, If you check the package antix-archive-keyring status, you should see Version: 20017.3.14 only! If you see Version 20017.3.14 and 20015.3.14, it will not work properly!
