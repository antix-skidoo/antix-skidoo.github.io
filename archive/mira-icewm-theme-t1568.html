I discovered by accident a necessary GTK engine for this theme is pixbuf. Sen7 did not list this as a requirement but I noticed something seemed a little flat. apt-get install gtk2-engines-pixbuf makes this theme come alive! Check out the difference in scrollbars, buttons, etc. in this screenshot compared with the screenie above. This also makes the theme more visible with subtle shading in different areas. This is truly an outstanding look if you like dark themes.
