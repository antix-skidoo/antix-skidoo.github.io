WARNING: For antiX users using antiX-M7.5 or earlier, please change your /etc/apt/sources.lst

If you want to dist-upgrade to antiX-M8 the use the antiX-upgrader2.sh script and read the instructions carefully.

If you do not want to upgrade to antiX-M8 and just continue as is, then you have 3 choices.

1. Run Etch (old Stable)
2. Run Lenny (new Stable)
3. Run Testing (Squeeze)

1. You have to have your sources.lst pointing to etch
2. Point your sources.lst to stable ot lenny
3. Keep stable and testing repos.

Option 1 is the safest.
