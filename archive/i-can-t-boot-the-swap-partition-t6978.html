A swap is not included in the Start menu.
Check it as root

Code: Select all

blkid -c /dev/null -o list
The entry in /etc/fstab looks, e.g. then so out

Code: Select all

UUID=016d8cde-618e-4236-9395-1a3b36fe3d1a swap swap sw,pri=1 0 0
