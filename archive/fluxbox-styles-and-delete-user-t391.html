As Anti said some styles use rounded corners...some do not. If you would like to enable or disable this feature for a particular style...open the directory for the specific style you want to alter.

Open theme.cfg or the main style sheet. Most styles have a logical flow with different sections grouped together. Look for the entries dealing with the menu...usually towards the top.

Look for this particular line or something close:
menu.roundCorners: TopLeft TopRight BottomLeft BottomRight
If you want to disable the rounded corners feature place a ! or # before the line. Save and exit.

If you want to add rounded menu corners to that particular style, add the above line. Look at the anti style for reference.
