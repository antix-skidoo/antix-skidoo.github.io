I always set up a separate data partition when I install. I find it best to keep my data away from the OS. You can set this up with gparted before you install while on the live CD. I normally leave /home in root and the mount the data partition in /home/data. This way if the OS craps out My data is safe. I can also sync this partition to a folder on my server for backup.
