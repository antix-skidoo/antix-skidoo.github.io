Por favor testem esse programa que instala um sistema num pendrive, com a opção de modo persistente (em que as modificações feitas no sistema persistem de um carregamento para o outro). O pacote anexado está traduzido para o português.

Ele foi feito para o antiX e o MEPIS mas deve funcionar com isos do Debian (e talvez com outros derivados, quem sabe...).

(pra quem não conhece, o antiX é baseado no Debian testing, otimizado pra usar poucos recursos do computador, localizado em português e esta versão vem com o kernel 2.6.32 do MEPIS).

Abra o terminal como root na pasta descompactada e

Code: Select all

./install.sh
No Ubuntu e pra quem tem o sudo configurado use simplesmente

Code: Select all

sudo ./install.sh
Isto irá copiar os scripts antix2usb-gui-br.py e antix2usb-br.sh para a pasta /usr/local/bin - portanto pra desinstalar basta remover esses arquivos.
