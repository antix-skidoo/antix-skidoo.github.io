you'll need to install pip. apt-cache policy commands just show the package, not install.

Code: Select all

sudo apt-get install python-pip python-dev
and then

Code: Select all

sudo pip install mps
sudo pip install glances
Note that glances is available in the repos already (synaptic)

Code: Select all

 sudo apt-get install glances
