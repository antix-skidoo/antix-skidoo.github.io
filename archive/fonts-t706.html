Hmm. The antix style is in /usr/share/fluxbox/styles/antix so you may need to copy it to ~/.fluxbox/styles and create a menu link to it (this is probably a good idea anyway if you want to include styles not on the iso). Then look for this line

window.font: Kristen ITC-9

Each style is different; e.g. some styles use a"global" font, which is usually declared at the beginning or end of a style file and is used in all parts of the style. Otherwise, in cases such as the antix style, the style definitions are usually grouped by category: menu, window, etc. Just find the section you want to edit and go for it.
