i assume that you want to install apps: simply open up synaptic, select the package you want and then select"mark for installation" after that click the apply button. or you can open up your terminal and become root with the"su" command and the type"apt-get install yourpackagename" (all without quotes).
and with the usb i assume that you want to install antix on a usb: here open up the control centre and then goto antix2usb to install on a usb.
