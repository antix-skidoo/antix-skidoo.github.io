If you mean to control the sound, pause / play, stop, next, previous etc you can add them to your hotkeys in vlc.
To do:
open vlc
menu --> run
type: vlc
press: ok
tools --> preferances
press: hotkeys
double click: the item which you would like to change (EG: for play, double click"play" then make the wanted key stroke. It may take a couple of times to get right __{{emoticon}}__ )
