In XP, go to START>SETTINGS>PRINTERS and right click on your printers icon. Enable sharing and XP will probably want to download a couple files before the printer is shared.

In anti-X, you will need to install smbclient if you want to access windows printer shares via SMB. You also need to install the hplip driver package.
Once you do this, install your printer by using the anti-X control center or via your web browser and localhost:631
