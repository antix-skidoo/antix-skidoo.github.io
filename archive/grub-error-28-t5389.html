grub-legacy, which is included in antiX (not MX) does not support booting in EFI environments. I'm less clear on the gpt which is what chromebook SSD's are formated by default.

you would need to use the seabios to boot grub-legacy.

If you have enabled seabios already, then I've got no other ideas.
