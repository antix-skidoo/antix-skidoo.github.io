Thanks for the upload Anti. Took me little while to break out the Asus Netbook and test it out.

I downloaded the uploaded message file from the provided link.

I downloaded it to /home/downloads folder
Used a suggestion from forum members and selected run from start menu and typed in pcmanfm and ticked the checkbox to run as root.
Entered my password when prompted.
PCmanFM opens up as root user.
I select File from the tab bar and tell it to open another tab.
I go to /home/downloads folder and display message folder
I select other tab and next use the up arrow to go to /boot/grub
I cut and paste new message from /home/downloads to /boot/grub
I get a message about do I wanna overwrite message in /boot/grub
I say overwrite all from the list pf buttons given me
Reboot and I have my new Mark Edelman Grub boot screen

Just posting this to thank Anti for the upload and anybody else reading this thread that wants to change there boot grub screen from the default black AntiX screen to the Mark Edlman one.

I am a happy camper as always because I run AntiX.
