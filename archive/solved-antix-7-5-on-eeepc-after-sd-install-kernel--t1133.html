If I remember correctly, if you do a hard-drive install to a usb stick, you need to use initrd too (I guess the same for the sdhc card). The installer asks around about the install grub part.

You may need to edit the grub/menu to point to the UUID rather than /dev/sda or /dev/hda.

See the excellent sidux manual for help how to do this. (It works in antiX too)
