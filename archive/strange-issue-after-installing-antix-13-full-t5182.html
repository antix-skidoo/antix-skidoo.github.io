spacefm by default unmounts when it exits. this is inconvenient default in my opinion.

you can change it though. go to Devices>Settings>AutoMount and uncheck"Unmount on exit".

If you use the rox desktop instead of the spacefm desktop, there is an option commented out in the startup file for icewm to run spacefm as a daemon in the background. That might be useful for you as spacefm will handle the mounting and everything will appear in rox as normal.
