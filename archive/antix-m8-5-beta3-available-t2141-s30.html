anti, I don't know if this is fixed in antiX (I ruined my installation while installing windows 2k). But the exoodles script as downloaded from the net has a minor error: in the codecs section the description is in a wrong order (in the description ffmpeg is #1 and flac is #2 but the commands are the other way round) and that confuses people.
The correct is (lines 692 to 695):

Code: Select all

${Comm}1${Stand}  - ${Comm}flac${Stand} Installs program to allow encoding of flac files.
${Comm}2${Stand}  - ${Comm}ffmpeg${Stand} ffplay multimedia player, ffserver streaming server
     and the ffmpeg audio and video encoder. Supports most existing file formats (AVI,
     MPEG, OGG, Matroska, ASF...) and encoding formats (MPEG, DivX, MPEG4, AC3, DV...)
I annexed a corrected file.
