I'm using the gtangish icon set from the 7.5 iso. (The folders are actually supposed to be black, but for some reason it looks like they stay blue while in /usr/share/icons. When in ~/.icons, the seem to turn black.)
Also where do you change the icons for the start menu and the show desktop?
Icons for"start menu" and"show desktop" are in ~/.icewm/themes/themename/taskbar. Start button is usually named"icewm.xpm" and show desktop is usually named"desktop.xpm".
And if I make changes how do I save them? For instance if I change the icons on the toolbar for the terminal, how do I save it so I can come back to it, if I try another theme?
If you want to have taskbar icons that don't change according to new themes, edit the file ~/.icewm/taskbar. Just include the path to the icons you want to use.
