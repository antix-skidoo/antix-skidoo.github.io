I recently played around with changing the icon theme in rox-filer. Here is a screenshot showing several different styles of icons in rox-filer.



I discovered there was very little documentation regarding the process. Here are some of the things I learned.

1. Most documentation tells you to add new icon themes to the ~/.icons file. For us antiX users the path is incorrect and new themes need to be placed in the /usr/share/icons folder. The rox-filer user manual can be found at:

2. Some of the stock icon themes do not work in my antiX installation. These include"hi-color","lo-color" and others.

3. To change an icon theme, right-click on the background of an open rox-filer window. Choose"Options", then"Types" on the left vertical menu. Clicking on the up or down arrow on the box to the left of"Icon Theme" will show you the choices available.

4. After changing your theme, close open rox-filer windows and re-open rox-filer for the theme to apply across all files.

4. Finding icon themes compatible with rox-filer is a hit and miss proposition at best. I've had success at the following sites:

The rox wiki is badly out of date and has many dead links. Some of the themes I've found at the sites listed in the wiki work....some don't.

You may have better luck. If you find any other nice sites with workable themes, add them to this thread.

6. To download a theme, use a tip Anti has mentioned in the past. Drag your download link from iceweasel into an open rox-filer window. The file will download into that destination without all the annoying popup windows asking if you really want to do this, etc..

5. For you fellow newbies, theme files come in a number of compression formats. To unzip or decompress the downloaded file open a terminal.

For a file with an ending of tar.gz type in the terminal........tar -xzf /home/(your username)/(your filename) ...substituting in your proper username and appropriate filename.

For a file with an ending of bz2, type.....bzip2 -d /home/(your username)/(your filename)

For a file with an ending of tar.bz2, type....tar -xjvf /home/(your username)/(your filename)

I would be interested in hearing your experiences with rox-filer customization.
