drg wrote: Question:

You could use antiX-core without any X (to keep RAM down) and only use cli apps eg for music (moc), torrent server (rtorrent), rss (newsbeuter), chat (irssi), email (alpine or mutt), net browsing (links2), video (mplayer), file manager (mc).

As soon as you install X RAM will go up to around 25MB.

You may want X and fluxbox (and feh to managed background)

apt-get update
apt-get install xorg fluxbox feh
