1. Theme (or style)

unpack it to ~/.icewm/themes and select it from icewm: start>settings>themes

2. Wall

4. Icons

========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========

Unpack it to ~/.icons. Then edit ~/.gtkrc-2.0 as follows:

Code: Select all

gtk-icon-theme-name="ALLGREY"
Note: If you change your gtk2 theme, you will need to re-edit this file.
