I tried firehol, guarddog, and finally firestarter. I installed it with apt-get and launched it as root. My needs are basic, so i had no problems, ShieldsUp gave me a perfect 'stealth' (all squares green).

# firestarter

This the order i did it:
1. Press the Forward button
2. Checked the box: IP address is assigned via DHCP
3. Forward
4. Next screen, left blank, i got no LAN. Press Forward
5. Check Start Firewall Now - Press Save.
6. From the menu choose Preferences
7. You'll see the status, click on Edit -> Preferences
8. I left Interfaces blank
9. Events -> Skip reduntant entries
10. Policy -> Apply policy changes immediately
11. Firewall -> Start/restart firewall on DHCP lease renewal.
12. Internet connected network device
13. ICMP Filtering
a. check Enable ICMP filtering box
b. Allow the following ICMP packets types
left it blank, did not mark any, the less packages, the more secure.
14. ToS Filtering
I checked the following boxes:
a. Enable Type of Service filtering
b. Workstations
c. Throughput
15. Advanced Options
I checked the following boxes:
a. Drop silently
b. Drop broadcast from external network
c. Drop broadcast from internal network
Press Accept and reboot
