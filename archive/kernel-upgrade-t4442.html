If you want to run antiX as a rolling release, use Testing (or Sid) repos and regularly upgrade your kernel. Make sure you keep at least one good working one just in case.

If you have old hardware and the kernel you are using works ok for you and you don't want to upgrade and just keep things as stable as possible, use Wheezy repo and default kernel,

Best to get the latest linux-image-3.9.5-antix.1-486-smp
Get its headers if you need to build nvidia drivers for example.

It should automaticall upgrade the initramfs and grub. If it doesn't, then after installing run these two commands in a root terminal.

Code: Select all

update-initramfs -k all -u -t 
update-grub
If grub is on another partition you will have to edit that menu.lst by hand.
