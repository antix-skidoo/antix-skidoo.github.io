toolbar.clock.font: sans-10:bold

the problem is that it doesn`t change the entries in the menu, but only for the titles.
I also added the entry in ~/.fluxbox/init. (i was looking for it, but couldn`t remember the term"overlay").

-
a few minutes later:
Once i knew what to search for i found another overlay-setup online:

Code: Select all

menu.frame.font: Bitstream Vera Sans-16
menu.title.font: Bitstream Vera Sans-12
toolbar.clock.font: Bitstream Vera Sans-12
toolbar.workspace.font: Bitstream Vera Sans-12
toolbar.iconbar.focused.font: Bitstream Vera Sans-12
toolbar.iconbar.unfocused.font: Bitstream Vera Sans-12
window.font: Bitstream Vera Sans-12
It did work.
