oldhoghead, while translating antiX's FAQ I thought that the paragraph regarding antix2usb could be improved (BTW, i'm translating slowly but i guess in about a week it'll be done). antix2usb can read antiX's iso file that was booted 'fromiso' but when booted from livecd/usb or an installed system it needs the iso file to install to the usb drive. So I put it like this:
Some Great Features in antiX.

* antiX2usb: For those familiar with the option in Puppy for persistent home via live USB, liveCD or fromiso: welcome! antiX now has a similar feature called antiX2usb and it is found in Control Center--> Disks.
You can use fromiso live to install antiX to a USB stick via antiX2usb and you can choose to have a liveusb with persistence. You can do the same from a live cd/live usb or an installed system using an antiX, MEPIS or Debian iso file. For a more detailed explanation see here.
