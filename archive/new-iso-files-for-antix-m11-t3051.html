New applications include

* pyneighborhood
* gvfs-fuse
* prism2-usb-firmware-installer
* speedcrunch
* toshset
* cdw

Before I go to store. I edited /etc/fstab so as to mount my Windows (hda1), Lighthouse Puppy Linux (hda6), and Fat32 Partition which holds old /home which I backed up to pull stuff off of (hda7). Rebooted and dragged partitions from /mnt over to thunars sidebar. Be sure to backup original /etc/fstab before editing original as a safety precaution.

My /etc/fstab

Code: Select all

# Pluggable devices are handled by uDev, they are not in fstab
/dev/hda3 / ext3 defaults,noatime 1 1
proc /proc proc defaults 0 0
devpts /dev/pts devpts mode=0622 0 0
/dev/hda4 /home ext3 defaults,noatime 1 2
# Dynamic entries below
/dev/hda1 /mnt/hda1 ntfs-3g auto,users,gid=users,dmask=002,fmask=113,relatime 0 0
/dev/hda5 swap swap sw,pri=1 0 0
/dev/hda6 /mnt/hda6 ext3 auto,users,exec,relatime 0 0
/dev/hda7 /mnt/hda7 vfat auto,users,gid=users,dmask=002,fmask=113,relatime 0 0
/dev/cdrom /media/cdrom udf,iso9660 noauto,users,exec,ro 0 0
/dev/hdc /media/cdrom udf,iso9660 noauto,users,exec,ro 0 0
