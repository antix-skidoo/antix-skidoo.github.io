    I found a solution on the dreamlinux forum.

    1. install alsa-oss and oss-compat packages
    2. su(x) to root and then type modprobe snd_mixer_oss
    3. try opening aumix from the menu
    4. You might need to reboot or do steps 5 and 6.
    5. /etc/init.d/aumix stop
    6. /etc/init.d/aumix start
