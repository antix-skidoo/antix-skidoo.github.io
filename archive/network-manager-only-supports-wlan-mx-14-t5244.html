Known Issues

The Sonics Silicon Backplane driver (ssb) conflicts with the wl driver (545388).

You have a Broadcom network card for eth0 AND a Intel PRO Wireless for wlan0. However from what you described, it appears that only wireless is working BUT is directed to the eth0. You need to correct that. Install the Broadcom drivers again with Synaptic.

The warning about"nsf" is nothing to worry about. The computer is noticing that you have a partition using nsf file structures on the hard drive. I get that on every boot-up with several partitions, some of which are using nsf (windoze). Don't worry. __{{emoticon}}__
