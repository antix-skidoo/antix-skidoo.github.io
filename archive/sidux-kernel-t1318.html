If the MEPIS kernel works well for you and you are happy with it, there is no reason to install the sidux kernel.
If there are some problems, the sidux kernel (now 2.6.28) may reolve them especially for new hardware.
The sidux kernels are upgraded regularly.
The sidux kernels are up-to-date.
Some have found the sidux kernel to boot slightly faster.
