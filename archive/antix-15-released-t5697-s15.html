Aleph wrote:1- I get this during boot:"A TPM error (6) occurred attempting to read a pcr value"
You should be able to get rid of this error message by blacklisting the tpm_tis module. On the live system you can do this with the boot parameter"blacklist=tpm_tis". On an installed system, try this instead:
echo blacklist tpm_tis | sudo tee /etc/modprobe.d/tpm_tis.conf
