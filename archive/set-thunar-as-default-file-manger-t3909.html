For toolbar icon. Open antixcc (crossed wrenches on toolbar) Edit Icewm Button Next. Then go to toolbar in geany.

Now to teach you something. Open a terminal Type in

Code: Select all

which thunar
you should get back in readout /usr/bin/thunar. That is the launch code.

Ok. in the geany text editor that is open and you are at toolbar section. Under
### Commonly used applications

Code: Select all

prog"Thunar" /path of icon found goes here  /usr/bin/thunar
For the Icon on Desktop. Just right click and edit by setting run action and edit name and set icon.

A left click on it should open thunar as user.
