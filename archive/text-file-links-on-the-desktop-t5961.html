the default rox desktop is not a folder in the tradiitional desktop metaphor sense. Its a"pinboard" of short cut links. So instead of copying files from the dvd to the desktop, you actually just made links. so when the dvd was removed, the links quit working.

If you are looking for a more traditional desktop, where the desktop is actually a folder, try the spacefm desktop.
