no problem. to be truly standard, you should copy the folder to /opt. That's where outside installed packages are supposed to go, but it really doesn't matter to much. If you leave firefox in your home folder, it will only be available to your one user. You can always make a .desktop file for firefox and put the .desktop file in /usr/share/applications. then update the menu and firefox should be in your menu.
