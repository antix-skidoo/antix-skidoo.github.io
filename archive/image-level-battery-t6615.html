1) desktopMenu, click"Control Centre" (i don't know what it would say in French)

2) in the Control Centre, click"Edit System Monitor(conky)" icon.
This loads .conkyrc (configuration file) ( geany /home/yourusername/.conkyrc ) into a text editor

3) in editor, scroll down to around line 210
##Battery examples##
##${color}battery: ${color}$acpiacadapter, ${battery_percent BAT1}%
##${color}battery:${color} ${battery}
#${color}ACPI Battery: ${color}$battery
#${battery_bar 11,0}
4) remove the start of line"comment" characters quoted above
5) perform File}}Save action in editor, then close editor

Afterward, after the conky utility has been stopped/restarted (when you logout, then login again)(or, in a terminal, command"killall conky" then"conky") you should see an onscreen"battery state indicator" displayed.
