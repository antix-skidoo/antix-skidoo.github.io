Lane,
Antix uses a metainstaller to install a lot of other programs. From the menu:
Applications>Accessories>metapackageinstaller

Root password window comes up and then the program starts. You can select different options. For HP printing:
Office>HP Printing.

I hope this helps and Welcome to Antix!
