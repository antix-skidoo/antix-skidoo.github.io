Ario made its debut in the 7.5 release. I may have had something to do with that decision.

I am still happy being the"maintainer" of the control center. Feel free to try some stuff out and send it to me. I'd like to share ideas and input with other users.

As far as icon sets are concerned, the control center and quit windows use icons from the gtangish black set that is currently on the 7.5 iso. We're going to use a different icon set in the next release, so we'll just use icons from that set or just keep the ones we like form the gtangish set. The hard part about using a new icon set is I couldn't get svg icons to work with gtkdialog so you need an icon set that includes 32x32 sizes.

Getting rid of wterm will not be a loss for me. I have not used urxvt, but I have used rxvt in the past. Roxterm has pretty nice tab features, making it nearly as good as GNOME's MGT (Multi GNOME Terminal), KDE's Konsole and XFCE's Terminal. If urxvt has these features, then perhaps Roxterm can go, otherwise I like it and have been using it with antiX, especially from Fluxbox.

In the Wifi space, ceni has always done the job for me. I advocate that we at least try going with it. As a console app it is one of the lightest Wifi apps around yet it works surprisingly well. We can get rid of the others, at least as a test and see what we gain or lose.

I have no strong opinions on other applications. For usability, I would like to see the Programs menu included in the main menu for both Fluxbox and IceWM. Hopefully this adds very little to the size of the image and improves usability.
