Right click the desktop and then choose"fluxbox,""file editing," and left click on"Keys." A window will appear (Keys). Start a new line at the bottom and enter

"Mod4 r :ExecCommand fbrun"
(w/o quotes)

Save and exit

Right click on the desktop and then left click on"Restart." This will restart fluxbox. Now, press the Windows Key + r and the command prompt should appear.
