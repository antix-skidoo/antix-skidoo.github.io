The shutdown command is 'sudo halt'.
For shutting down from IceWM's menu (Exit>Shutdown), edit ~/.icewm/preferences, look for 'shutdown command'. I believe it would be something like

Code: Select all

ShutdownCommand="killall wbar && sudo halt"
You can substitute 'sudo halt' for 'killall wbar && sudo halt' in exitantix.sh as well.

-edit- the same for the reboot command.
