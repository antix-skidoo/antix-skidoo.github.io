SERVER SOFTWARE INSTALLED

config-util script to setup an initial configuration specific to you after a fresh install

VNC--------------------------------------------------------
Configured in full to run using lightdm configuration

SSH--------------------------------------------------------
Set to standard debian install on port 22

ProFTPd----------------------------------------------------
ProFTPd is preconfigured to access
~/ for all members of the users group
~/Share for all members of the shares group
~/Website for all members of the Website group
root is allowed full login and viewing.

Apache2----------------------------------------------------
Apache2 is preconfigured for
/mail for squirrelmail
/debs for http directory of the preconfigured repository
files
/repo for the repository, the sources.list line
deb

this will be explained in the reprepro section
/logs for the webalizer log access.
/demo for live user demo's Website directory

running config-util can setup other users with website directories

DOVECOT----------------------------------------------------
Dovecot is preconfigured for any system user,
mail for all users is saved in /home/USER/Mail.
Your mail client should work with any of the following
settings

imap4, imap4s
server address: SERVER-IP-ADDRESS
server port: 143 for non-secure/tls 993 for ssl
server security: start-tls, ssl, none
username: your system username (for live: demo)
password: your system user's password (for live: demo)

pop3, pop3s
server address: SERVER-IP-ADDRESS
server port: 110 for non-secure/tls 995 for ssl
server security: start-tls, ssl, none
username: your system username (for live: demo)
password: your system user's password (for live: demo)


POSTFIX----------------------------------------------------
Postfix is preconfigured for any system user,
however it will only work for lan use.
If you go through the config-util it will setup the server with a proper domain for internet use.
Manually you need to view the server documentation file in ~/Desktop to fix

SQUIRRELMAIL-----------------------------------------------
Squirrelmail is preconfigured for any system user,
click the squirrelmail shortcut on the desktop or open

Procmail---------------------------------------------------
Procmail is preconfigured to recieve mail handed off from
postfix, run it through spamassassin, and place in the
appropriate mailbox (USER/Mail/inbox or USER/Mail/junk)
You should not need to change this.
