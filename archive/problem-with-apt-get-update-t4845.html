An idea that may not identify the cause but might help narrow down the scope of the problem.

Boot up live using a known good live media
Does apt-get update work?
If no the problem is not in your local system
If yes the problem is in your local system

From the live system copy one repo entry verbatim to the sources list in your installed system
Comment out all other entries in the installed sources list
Boot the installed system
Does apt-get update work?

If yes comment out the transferred live entry
Uncomment the only the equivalent entry in the installed sources list
Does apt-get update work?

If no the problem is related to the format of the entry or apt+friends
If yes uncomment a further entry and repeat the test.
