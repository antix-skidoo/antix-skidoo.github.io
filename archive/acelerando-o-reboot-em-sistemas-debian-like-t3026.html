Acelerando o reboot em sistemas Debian-like

Uma dica simples para acelerar o reboot em máquinas rodando Debian ou derivados. Para tal instalaremos o pacote kexec-tools.

Segundo a descrição, o pacote kexec-tools carrega o kernel na memória e, ao reiniciar, utiliza-o através da chamada de sistema kexec. Informações técnicas à parte, instale-o assim:

$ sudo apt-get install kexec-tools

A partir daí você nunca mais verá aquela horrível tela inicial de teste da BIOS quando reiniciar seu sistema Debian-like. Isso inclui a tela do Grub. Considero ideal para aqueles que usam apenas uma distro, ou mesmo para distros Debian-like rodando no VirtualBox.

Para aqueles que ainda tem aquele famoso S.O. da Microsoft em dual boot com Linux, ou mesmo uma outra distro, estejam avisados novamente que não verão a tela do Grub ao escolher"Reiniciar". A tela do Grub só será vista caso escolham a opção"Desligar". Caso não estejam satisfeitos com o funcionamento do programa, remova-o dessa forma:

$ sudo apt-get --purge remove kexec-tools
