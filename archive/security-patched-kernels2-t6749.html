Due to another exploit, we have had to patch our kernels.
Kernels patched will be 4.4.10 and 4.6.2 in 32 and 64 bit, standard and gnu.

They should automatically install with apt-get update && apt-get dist-upgrade.

You may need to install any proprietary drivers for the new kernel via smxi/sgxfi if dumped into a no desktop environment on reboot after installation of new kernel.

Before upgrading, please make sure you make a backup of your important files/folders and/or of the whole operating system via snapshot.
