so you need at add an entry in fstab.

Code: Select all

#Entry for /dev/sda3 :
UUID=e2f9c4ca-11a9-42ca-b996-e3d4b4a3de8e    swap    swap    defaults    0    0
save, then try

Code: Select all

sudo swapon -a
