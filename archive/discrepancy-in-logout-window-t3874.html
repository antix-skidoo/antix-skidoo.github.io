Depending upon which button is chosen in the Log Out window (exitantix.sh) different user input is required. I have tested on Shutdown, Reboot and Logout as these are the only ones I use. The tests were done in antiX-12-Base, within JWM.


Log Out Window Displayed
<TAB><TAB> (to select Reboot button)
<ENTER> (to activate the button)

Result
Machine reboots

Alternative
A single mouse click on the Reboot button reboots the system.




Log Out Window Displayed
<TAB><TAB><TAB><TAB><TAB> (to select Shutdown button)
<ENTER> (to activate the button)

Result
Machine shutsdown

Alternative
A single mouse click on the Shutdown button shutsdown the system.




Log Out Window Displayed
<TAB><TAB><TAB> (to select Log Out button)
<ENTER><ENTER> (to activate the button)

Result
User logs out (after 2x<ENTER>)

Alternative
Two mouse clicks on the Log Out button logs out the user.
OR
A single mouse click on the Log Out button plus moving the mouse pointer outside the Log Out window logs out the user.
