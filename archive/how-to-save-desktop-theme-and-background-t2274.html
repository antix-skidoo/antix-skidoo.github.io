That should be automatic, vinoman. When you choose an IceWM theme or a Fluxbox style they should keep it.
As for the desktop background, if you use Control Center>Choose Wallpaper it should remind the chosen one in both IceWM and Fluxbox. While it was supposed to work if the images are in a different directory than ~/Wallpaper, when I tried it didn't. So just keep your wallpapers in ~/Wallpaper.

As you may know, if you use ROX-Pinboard then the desktop background is set differently. Also, just mentioning, if you want to use Conky with ROX-Pinboard you must edit ~/.conkyrc
-change alignment_top_left for alignment_top_right
-set own_window to yes
otherwise comment Conky's line in ~/.icewm(.fluxbox)/startup so it won't start.

You're welcome. Just don't comment the nitrogen lines in ~/.icewm/startup. It is those lines that load the saved background when you login (you can comment them if you set a fixed background for the theme you're using and you don't want to change it anymore).

The saved backgrounds are in ~/.config/nitrogen/
