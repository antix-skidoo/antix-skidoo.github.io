By default antiX-12 places an audio volume control in the taskbar for IceWM but not for JWM.

To add it requires a simple edit.
antiXCC-->Desktop Tab-->Edit JWM Settings-->Startup Tab
add the following:

Code: Select all

...
# Uncomment to place an audio volume control in the JWM tray (taskbar)
volumeicon &
Alternative
IceWM includes a sleep command that was unnecessary on this system, so was omitted as it only slows down loading.

Code: Select all

...
# Uncomment to place an audio volume control in the JWM tray (taskbar)
sleep 3 && volumeicon &
