    I got this on an old antiX-M7.5 ugrade too.
    The way I fixed it, is to remove these 3 files
    wicd.postinst, wicd.postrm, wicd.prerm,
    from /var/lib/dpkg/info

    Then apt-get -f install

    (You may need)
    apt-get install --reinstall wicd
