In case anyone is running a super lightweight setup of antiX and wants a lighter work processor than Abiword, Ted will install and run if you download and install these files from the Debian site:

ted-common_2.16-5_all.deb

ted_2.16-5_i386.deb

Note: the version listed here is *not* the gtk version (which will not install on Lysistrata--at least not on mine), and it is not the 2.17-1 version listed in the current repos (which will install but has a bug preventing the opening of any documents).
