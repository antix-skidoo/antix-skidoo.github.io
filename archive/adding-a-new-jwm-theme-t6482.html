For my own use I created a new JWM theme called"tangerine" and added the script to the folder ~/.jwm/themes. I wasn't able though to update the theme menu by clicking the themes menu option"Update theme list". Obviously the required script update-jwm-themes.sh is missing in antix-16.

Well, finally I found out that I could activate my new theme by simply adding it manually to the ~/.jwm/themes-list.
