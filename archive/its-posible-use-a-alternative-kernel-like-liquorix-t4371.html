With antix-snapshot - it won't work without a lot of hoops.

The good news is that it does work using the refracta-snapshot and refracta-installer. I just tested it.

I installed latest 3.9.4 liquorix kernel, rebooted into it and removed the antiX one.

I then downloaded refracta-snaphot base and gui and the 2 refracta installer debs from here:


========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://distro.ibiblio.org/refracta/testing/"
linktext was:"http://distro.ibiblio.org/refracta/testing/"
====================================


dpkg -i *.deb followed by apt-get -f install to pull in all the dependencies ie the live app it uses and xterm.

Then run the gui refracta-snapshot to create a bootable live iso which tou can then install using the refracta-gui installer.

99.9% perfect!

Added: I have added the 4 refracta debs to the antiX repository so you should be able to install then (and dependencies) with apt-get.
