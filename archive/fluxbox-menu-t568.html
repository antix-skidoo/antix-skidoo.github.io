You can use the debian fluxbox menu and it will 'autonatically' add debian apps to the menu, though it is ugly IMO and it doesn't add the Mepis/antiX ones.

Simply, edit this line in the .fluxbox/init file to read

session.menuFile: ~/.fluxbox/fluxbox-menu

Check that .fluxbox folder has a fluxbox-menu file (it should have) If not you'll have to copy the one from /etc/X11/fluxbox/

Restart fluxbox (from menu)
