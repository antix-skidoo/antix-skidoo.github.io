This will set the keyboard for the session:
$ setxkbmap tr

For permanent Turkish keyboard (for latest antiX)

In
/etc/default/keyboard (if you have it)
XKBLAYOUT="tr"

or in
/etc/default/console-setup
XKBLAYOUT="tr"

or in
/etc/X11/xorg.conf
XKBLAYOUT="tr"

To set the keyboard immediately, type in a terminal as root

Code: Select all

setxkbmap tr
To set it definitely you must edit either /etc/default/console-setup or /etc/default/keyboard. It depends on how up-to-date your system is, so if you want you may post here the contents of those two files for us to see what to edit (basically you'll have to change the layout from"de" to"tr").
