1) make sure the"sis" kernel driver is loaded:

Code: Select all

lsmod | grep sis
If it is not loaded then run:

Code: Select all

sudo modprobe sis
2) Remove this section from your xorg.conf:

Code: Select all

    SubSection"Display"
        Modes"1280x1024""1333x768""1024x768""800x600"
    EndSubSection
or change it so the first resolution is the native resolution of your screen like:

Code: Select all

    SubSection"Display"
        Modes"1280x800""1280x1024""1333x768""1024x768""800x600"
    EndSubSection
3) Then restart X by going to"Logout" in the main menu and then choosing"logout" again in the desktop-session-exit window. You will then have to log in at the SLiM login window.

Note that the"sis" kernel driver is not automatically detected on the Live system.
