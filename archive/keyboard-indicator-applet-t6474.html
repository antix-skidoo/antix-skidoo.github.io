In Antix-15 there was this nifty keyboard layout indicator applet. It showed a small icon on the taskbar with a different symbol for each active keyboard layout (visually appealing in the form of a flag - I liked that touch). That applet seems to have disappeared, or at least it's not active in my fresh Antix-16 64 bit installation.

Any suggestions on how to re-enable it? Or are there alternatives to consider? I need somehting of the sort since I routinely use three different keyboard layouts.

After updating my Antix-15 system, the applet had also disappeared there. So I suspect it has maybe been removed.

TheRe is a commented entry for it. With the control center go to session tab>user desktop session and click on startup tab. Uncomment the entry you want.

Thanks for showing me the way. That's solved. Strangely in Antix-15 I found no reference to fbxkb (which is the name of the applet used), not in User desktop session, nor in Global desktop session. Adding the entries there didn't have any effect. I had to use tab>Desktop, Edit fluxbox settings and add"fbxkb &" to the fluxbox startup configuration file. That solved my problem there (where it was actually more urgent).
