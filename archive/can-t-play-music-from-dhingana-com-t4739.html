I got it to work here in antiX-13.2 and iceweasel (firefox) 22.0.

In the iceweasel (firefox) menus I went to:

Code: Select all

tools --> add ons --> extensions

   flashblock --> preferences
       [ ] Enable Flashblock
and UNCHECKED"Enable Flashblock". Simply disabling flashblock via the button next to the"preferences" button did not work.

A better/safer way is to move to the Flashblock"whitelist" tab and add"
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.dhingana.com"
linktext was:"www.dhingana.com"
====================================
" to the whitelist. I tried this and it also worked. This only disables flashblock for that one site.
