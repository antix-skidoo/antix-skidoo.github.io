Move this

Code: Select all

title Debian GNU/Linux, kernel 2.6.35-5.dmz.1-liquorix-686
root (hd0,1)
kernel /boot/vmlinuz-2.6.35-5.dmz.1-liquorix-686 root=UUID=e293d5f6-20bb-469a-8a91-7b73fb64764d ro nosplash quiet acpi_osi=Linux
initrd /boot/initrd.img-2.6.35-5.dmz.1-liquorix-686
to above the first title antiX at sda2, kernel 2.6.32-1-mepis entry and below the gfxmenu /boot/grub/message.

You can also change the title if you wish.
