antiX comes with 3 apps that can work as wireless managers - wicd (Connect Wirelessly), ceni (Configure Network Interfaces), and mnetwork (Manage network Interfaces). All these can be accessed from the antiX control center on the network tab.

Anyway, I have not connected to wireless since upgrading to 8, but with 7, I could never connect using anything but ceni. I prefer rutilt for connecting; it can be installed using synaptic or apt-get.

In a console you can type lsusb as root, and post what your dongle is, in case you need help configuring the device. Sometimes wireless drivers need to be blacklisted for others to work.

PS. I just looked at these in 8 and they all have some improvements, so maybe this time... __{{emoticon}}__
