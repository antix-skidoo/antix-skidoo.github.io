    failed to install Qarte
    You attempted to install it via the"gdebi" utility?
    If so, gdebi would have displayed explicit reason(s) for the failure.
    Typically failure is due to absence of one or more packages required by the to-be-installed program
    and/or presence of incompatible builds/versions of installed package libraries.

    If gdebi is reporting"dependencies could not be met", post the details here and hopefully we can find a solution.
