Greetings!

which version of antix-mx did you install (I'm going to assume MX-14 32 bith NON-PAE).

My first guess is that you need to update flash player. There are some caveats, as not all CPU's support the latest flash. If you can post the output of

Code: Select all

inxi -F
that should give us the info to determine if you can run the later flash. If so, you can use MX Flash Manager (in the menu) to update flash to a later version.

***edit*** Actually, you can use MX Flash Manager anyway, and it should update you to the latest Flash available for your cpu. The inxi -F info would still be helpful.
