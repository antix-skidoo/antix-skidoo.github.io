Do this:

Code: Select all

dpkg-query -l | grep"ii" | awk ' { print $2}' | awk ' BEGIN { RS="\0";FS="\n"} { for (i=1; i<=NF; i++) s=s"" $i } END { print"apt-get install" s}' > installed.sh
Make the file executable and change apt-get install to apt-get install --reinstall then run the script. Hopefully this will rebuild the /var/lib/dpkg/info files
