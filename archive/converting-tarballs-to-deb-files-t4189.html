Would it be easier to just convert tarballs to .deb instead of following the steps to install the tarball?

There is only two steps to install .deb

Code: Select all

wget (file url)

dpkg -i filename.deb
Rather than:

Code: Select all

wget (fileurl)
mkdir ./BUILD
./Configure
su 
Make install
