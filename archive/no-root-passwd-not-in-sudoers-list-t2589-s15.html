For sidux keyring you must have its repo enabled (sometimes you have to add debian/ to the end of the address in that list of mirrors) and have apt-get updated. (I don't leave it enabled, only used it to add some app and then disabled it again)
You must run the commands to add the keys (VB and Google) as root.

Those commands for vlc and perl-tk are to get their build dependencies, is that what you want? If so you must enable the sources repo (the one starting with deb-src).

Also for you to edit /etc/sudoers you must login as root in the terminal and run the command visudo.
