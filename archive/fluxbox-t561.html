I'm assuming the fluxbox menu you download for regular Mepis will be different from the menu that anticapitalista configured for antiX. I'm not sure why you'd get duplicated entries, but if you open fluxbox menu you'll see which menu entries got doubled up. Delete what you don't want, but save a copy first.

In the antiX version, you right click, go down to fluxbox, over to file editing, and then menu. I'm not sure where this is located on the generic fluxbox menu (probably easy to find). Otherwise, you can just go to the fluxbox directory and find it there (a hidden file in your user directory).
