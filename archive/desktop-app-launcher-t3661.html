I think I have wbar and wbar-config 2.2.2 or similar as wbar 3 has a bug with xfdesktop. Anyway I used wbar-config, pressed the add new entry button, made the name"menu" made the binary"slingshot" and made the icon"/usr/share/pixmaps/xfmenu1.png" or close to that. Pressed add and then refresh and there it was.

I will post the exact steps tomorrow and the manual configuration as well.

EDIT
Version of wbar 2.2.2
With wbar-config
menu --> run
type: wbar-config
press: enter / return
press: new
Type:"Menu" under title
type:"slingshot" under command
click:"..." under Icon
Choose: the icon you would like
Press: open
Press: accept
Press: reload

With text editor
menu --> run
type: leafpad ~/.wbar
press: enter / return
go to: line 4
press: enter / return
type: i: path to icon
press: enter / return
type: c: slingshot
press: enter / return
type: t: menu
* NEW ENTRY SHOULD LOOK LIKE *

Code: Select all

i: /usr/share/pixmaps/xfce4_xicon1.png
c: slingshot
t: Menu
press: enter / return
file --> save (ctrl + s)
close
If wbar is running, right click on it.
If wbar is not running,
menu --> run
type: wbar
press: enter / return
