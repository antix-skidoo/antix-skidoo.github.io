Vronsky,

I saw your post and Kano's reply on the Kanotix forum.

If you are still around, try these steps.

1. Open a terminal, sux and give your root password.
2. Type in rox
3. Navigate to /etc/modules file
4. Right-click on file>Set Run Action>Type leafpad in the box. (make sure you leave a space between the 'd' and the" >Use Command
5. Now left-click on the file and copy/paste the snd info.
