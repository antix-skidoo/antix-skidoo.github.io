the gpg key for the antix repo needs refreshed.

follow this procedure:

1. open synaptic >repositories and disable the antix.daveserver.info repository.

2.. reload sources

3. enable the antix.daveserver.info repository

4. reload sources

5. find antix-archive-keyring and update that package.

6. once that package is up to date, you can reload all sources and your error should go away.
