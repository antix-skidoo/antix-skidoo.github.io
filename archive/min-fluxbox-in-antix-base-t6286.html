The min option starts only the minimal number of items for a desktop. So all startup is disabled and only the window manager is started. From there you should be able to modify desktop-session.conf (to enable / disable options), desktop-session startup (if you leave it enabled in desktop-session.conf) , and the window manager startup (~ / . fluxbox / startup in your case) . Remember that min is minimal so everything is pretty much disabled so if you like the min- option then disable everything and add only the stuff you would like.
