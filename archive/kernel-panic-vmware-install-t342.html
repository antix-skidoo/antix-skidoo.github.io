I am a"distrohound" and I finally decided to give this one a try. I do my testing on VMWare Server. I downloaded both base and full versions today for a trial.

The install worked fine for both, but each failed on reboot with the following message,"kernel panic - not syncing : VFS: unable to mount root FS on unknown-block (0, 0)"

I tried reinstalling the system and reinstalling grub using the Mepis system tool without success.

After a little Googling, I inserted the virtual cd back into the virtual machine and reinstalled grub again. This time I enabled initrd, rebooted and all is well!

Now for some testing and I hope this may help someone else. __{{emoticon}}__
