This one is bigger than previous test isos not because I have added some more apps, but because I have made localisation to be much better.
As usual, give this a run out particularly testing wireless, wired network connections, general feel and also the localisation.

On installation, you can choose your locale and keyboard. This should actually work ok for root user, but it needs a couple of extra tweaks to get the whole system in that language.

eg Spanish language. If you choose when installing Spanish utf-8 (recommended), then once installed you should have this line in /etc/default/locale

LANG=es_ES.UTF-8

Now here are the extra tweaks.

1. Edit /etc/profile to add the line, below the other export entry

export LANG=es_ES.UTF-8

2. As user in a terminal (This updates icewm menu)
update-menus

3. As root in a terminal (This updates fluxbox menu)
update-menus

4. Logout/login to a menu with Spanish entries and most apps with Spanish menu bar.

5. To get antiX control centre and exit scripts, edit fluxbox menu and icewm menu and icewm toolbar.
Change antixcc.sh to antixcc-es.sh and
exitantix.sh to exitantix-es.sh

Edit: You will notice that a newer MEPIS kernel will be installed with an upgrade. This should be ok (There were problems in live mode).
