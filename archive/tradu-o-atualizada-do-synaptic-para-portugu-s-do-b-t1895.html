Foi atualizada a tradução brasileira do Synaptic. Em algum tempo ela deve ser incorporada no repositório. Quem quiser atualizar já, baixe o arquivo anexo, descomprima-o, renomeie o arquivo /usr/share/locale/pt_BR/LC_MESSAGES/synaptic.mo para synapticbackup.mo, p. ex. e, no terminal:

Code: Select all

msgfmt /caminho/do/seu/arquivo/pt_BR.po -o /usr/share/locale/pt_BR/LC_MESSAGES/synaptic.mo
Então examine se está tudo bem com a tradução no Synaptic e pode apagar de vez o synapticbackup.mo

* este arquivo foi feito com o pacote do Synaptic 0.62.7, eu vou examinar o arquivo da versão 'stable' e, se for diferente, eu posto aqui uma tradução específica para ele *
