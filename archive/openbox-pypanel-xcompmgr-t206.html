tip for enabling composition via xcompgr in Neomagic 256AV cards (whopping 2.5MB RAM __{{emoticon}}__ ). Paste this somewhere in your xorg.conf

Code: Select all

Section"Extensions"
       Option        "Composite"  "Enable"
 EndSection
Points to notice:
menu shadow
true transparency

CAUTION: Don't expect beryl/compiz fusion performance. This thing is SLOW. This is not really a fancy shot, might update with something better.
