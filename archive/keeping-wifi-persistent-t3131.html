use wicd, it is included in antiX full.

you will need to run
-"wicd-client -n" or use the control center --> network --> connect wirelessly.
- click on prefferences in the wicd network manager
- poke in the wireless interface as ceni says that it is and press ok
- press refresh and when your wireless access point appears press properties
- punch in the properties that you want to stick and press ok
- check off automatically connect and press connect.
- close the manager and forget about it
