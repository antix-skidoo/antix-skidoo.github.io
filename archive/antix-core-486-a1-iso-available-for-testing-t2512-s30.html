Glad you're up and running with core.
Core offers even more flexibility than base as it is really a live, installable debootstrap base with some apps removed and others added ie locales. Once installed you have the base for a cli only box if you have very low RAM and CPU power as drg has (see this link antix-m8-5-486-on-old-hardware-t2336.html ) or to install whatever desktop environment and apps you choose from the Debian repos (Testing or Sid or -when squeeze goes final- stable)

The only apps from MEPIS is the kernel and the mepis-gfxboot file, both can be replaced if user wishes to have a 'pure' Debian.
