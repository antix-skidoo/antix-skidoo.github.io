greyowl wrote:With Ice and JWM, there is a menu in the right hand corner of the task bar.
You mean the LEFT hand corner?

Well, there is a way to add a menu to the toolbar of the pre-installed fluxbox version.

Open the file ~/.fluxbox/init in an editor, find the line that starts with"session.screen0.toolbar.tools:" (here line number 30) and add"RootMenu" at the desired position:

Code: Select all

session.screen0.toolbar.tools:Â Â Â  RootMenu, workspacename, prevworkspace, nextworkspace, iconbar, systemtray, clock
If you really prefer your menu in the RIGHT hand corner, append"RootMenu" to the end of the line.

You will find the menu represented by a simple triangle, though. I have no idea if it is possible to use a nicer icon instead.
