The fbxkb package gives you a little GUI that goes in the system tray. But the Debian version is a little broken. We have a fixed version (0.6.2) in MX-14 but I'm not sure how you can get just that package. Others might know. There is also the xkmap script. If you Google(setxkbmap GUI) then you might find others. There is a plugin for gkrellm that has this functionality but I don't know if that plugin is available in Debian.
