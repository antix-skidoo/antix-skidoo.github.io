This is the app OU812 made/modded to update the menus. If you look at ~/.icewm/menu you'll see this is the command called when we click on Menu>Desktop>IceWM>Update menus (auto)
It doesn't use the programs file anymore, now it uses the desktop files in /usr/share/applications.
/usr/local/bin/icewm_menu2.sh runs ~/.icewm/icemenuyap2.py pointing its output to ~/.icewm/application. Then the menu reads it.
