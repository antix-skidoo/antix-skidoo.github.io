I made a multi boot usb with antiX 13.2, 32 bit, 64 bit, mx 14 pae, and antiX 14.

I did this by mounting the isos.
Making 2 partitions on the usb (1 for data and 1 for the distrobutions)
In the first partition I made 4 folders, one for each distribution.
(antix-13.2-32, antix-13.2-64, mx-14-pae, antix-14)
Then I copied the contents of each iso to their respective folders.
Then installed syslinux to the mbr
Then edited the syslinux.conf file adding an entry for each distribution, the key to the successful boot of each was the cheat code of bdir=/path/to/distrobution/folder. Otherwise I would get kernel mismatch.
