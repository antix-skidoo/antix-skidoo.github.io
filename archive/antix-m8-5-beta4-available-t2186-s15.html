antix2usb-gui.py (br version)
anti, I'm glad to say that it's fixed. Here's how it is (lines 197 to 226):

Code: Select all

        # The ISO
        args +="-I %s" % self.isoFile
        
        #lang=pt_BR
        args +="-c lang=pt_BR"

        # The target device
        args +="-D /dev/" + self.device +""

        # bootable partition
        args +="-f %s" % self.fileSystem
        args +="-s %d" % self.partitionSize
        # data partition
        args +="-F %s" % self.fileSystem
        # persistence
        if self.persistence:
            args +="-p"

        if DO_DEBUG :
            args +="-d"

        # lang=xx
#        lang = os.environ['LANG']
#        if lang:
#            lang = lang[0:2]
            #print lang
#            args +="-c lang=%s" % lang
            
        cmd = ANTI2USB_SCRIPT +"" + args
        return cmd
