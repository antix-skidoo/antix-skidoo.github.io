anti,

i've got the fix for steamtuner. the mpv command for youtube is incorrect. You need to change the key value for

Code: Select all

video/youtube
in ~/.config/streamtuner2/settings.json

to

Code: Select all

"mpv %srv"
in other words, deleted the $(youtube-dl -g stuff. Its wrong.

I couldn't get the settings to take affect when trying to change the command in the gui field, even though it *looked* like the field changed. I had to edit the .json file.
