In a terminal, type su then your password to get root status. Then use leafpad to edit the file. Again in the terminal type leafpad /etc/apt/apt.conf
This will bring up leafpad (an editor). Change the size to whatever you want (or was recommended). Save and that's it. Maybe reboot. Not sure about that though.
I hope this helps.
