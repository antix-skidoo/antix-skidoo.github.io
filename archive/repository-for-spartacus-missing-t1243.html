Unfortunately antiX-Spartacus is old. It appears that the Mepis repositories no longer are used (possibly the same for the Ubuntu ones Dapper).
You could try changing the Ubuntu ones to Edgy or Fiesty, comment out the Mepis ones, reload synaptic and read very, very carefully what synaptic will do when you want to install an app.

Personally I would use apt-get, as root, in a terminal to install as it lets you know what will happen if you install and if it will completely hose it or not.

IMO, do not do a smart upgrade/apt-get dist-upgrade or even apt-get upgrade.
Just install individual apps eg apt-get install nameofapp.

Good luck
