Type in terminal

Code: Select all

which thunderbird
That shows the launch path for Thunderbird.
You can add thunderbird to the Icewm Taskbar or Fluxbox Menus via the Control Center in AntiX.
I am on my LXDE Laptop right now so I can't show you how to edit via control center.

Conky remembers in AntiX 11 from Icewm to Fluxbox.
