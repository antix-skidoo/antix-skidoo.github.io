    Have a look in the xorg.conf.live file to see if it looks something like this.

    Code: Select all

    Section"Monitor"
    Identifier"Monitor0"
    Option"DPMS""true"
    HorizSync    30-75
    VertRefresh  55-70 
EndSection

Section"Device"
    Identifier"Device0"
    Driver    "vesa"
EndSection

Section"Screen"
    Identifier"Screen0"
    Monitor"Monitor0"
    Device "Device0"

    SubSection"Display"
    EndSubSection
EndSection
    If it does, then rename it to xorg.conf (as root user) and reboot.
    It should boot to a vesa desktop
