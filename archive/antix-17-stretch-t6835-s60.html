reverseDog wrote: SpaceFM actually provides three ways of drag&drop:
(1) Simple drag&drop will move a file.
(2) If you want to copy instead just press and hold down the the CTRL key while dragging (but let it go before dropping).
(3) CTRL+SHIFT while dragging creates a symlink.
Methods (2) and (3) work best when you start pressing the key(s) before grabbing the file.

This holds for drag&drop to/from the desktop as well as between any other folders (in different tabs/panels/windows of SpaceFM) – but doesn't seem to work between devices where you always end up in copying.

If desired the default drag behaviour can be changed: View -> Preferences -> Interface ->"Default Drag Action".
