If you want a change to the clock to survive style changes, then put the modification in the ~/.fluxbox/overlay file. This will change the clock

Code: Select all

toolbar.clock.font: sans-10:bold
the example uses the sans font, size 10, bold style. remember to restart fluxbox (there is an option in the fluxbox setting menu) for the changes to take affect. You don't have to close your text editor to restart but you do have to save the file.
