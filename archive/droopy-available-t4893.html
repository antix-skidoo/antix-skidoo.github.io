What is it?
An impromptu way to move files over a network to or from Linux, Mac, Windows, Smart Phone etc


Why?
It's easy to use, convenient, lightweight and tiny.
No additional software is required on the remote system other than a web browser.


Want to know more?
It's included in the antiX online FAQ.

How to install it
It's in the antiX Stable repo and is called droopy-antix.
Install it as usual via Synaptic, apt-get, or your preferred method.
After installing refresh the menu via, antiX Main Menu-->Update Menu


How to set it up
Copy the hidden folder
  • /etc/skel/.config/droopy
To the hidden folder
  • /home/USERNAME/.config/
Create the folder to serve
  • /home/USERNAME/Public
Note: The name is case sensitive and must be named Public

Drop a few files into the Public folder

Start Droopy via the antiX main menu.

Note
When installing from the antiX-15 repo the manual copying of hidden folder and creation of the Public folder are not needed. Everything is set up automatically when Droopy is run for the first time
