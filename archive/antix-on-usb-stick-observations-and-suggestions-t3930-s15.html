If I had read your last post carefully, I would have seen you were reporting about two new issues:
  • The amount of time it can take to check the filesystem varies, and
  • It always does the save even when you tell it not to!
It is possible the extra question asked in manual and semi-automatic mode is what is making them go slower. The scan of the rootfs file system to see how much stuff is on it will saturate the cache on a system without much memory so it will take longer for the next message box to appear.

It should be significantly quicker in command line mode. For example, you could trying running:

Code: Select all

$ sudo persist-save --cli
from a terminal window.

The second issue is a glaring bug.
