Does it show up when you open the network manager, if so add wicd-gtk to the icewm startup by:
menu --> run
type: add-start
press: ok
follow allong
restart icewm

or

menu --> run
type: leafpad .icewm/startup
add: wicd-gtk &
save and close
restart icewm
