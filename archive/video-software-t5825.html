Install Howto

Add the following line to / etc/apt/sources.list:

deb
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.deb-multimedia.org"
linktext was:"http://www.deb-multimedia.org"
====================================
jessie main

Update the package index:

# sudo apt-get update

Install GPG key of the repository:

# sudo apt-get install deb-multimedia-keyring

Install dvdstyler deb package:

# sudo apt-get install dvdstyler
Uncomment .deb-multimedia in your sources list. apt-get update first. Then install dvdstyler that way. Then comment out .deb-multimedia. apt-get update again. That way you won't break anything. Reason is. Leaving deb-mutimedia enabled breaks stuff in AntiX and MX-14.
