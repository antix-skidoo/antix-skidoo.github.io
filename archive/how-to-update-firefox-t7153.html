Welcome to antiX.

antiX-16.2 uses the default firefox-esr version in Debian which will work with old PIII boxes, unlike the newer version which needs a P4 minimum. Debian upgrades the version (ESR) we default with with security fixes and they will show up in synaptic/apt-get when a new upgrade is needed.

If you want the very latest firefox you should use the meta package installer app in the control centre. Note latest firefox is a RAM/CPU hog.
