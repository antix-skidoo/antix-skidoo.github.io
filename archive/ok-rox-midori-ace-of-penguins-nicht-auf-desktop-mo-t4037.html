    Unter icewem/rox kann ich für Midori und für die Ace of Penguins Spiele zwar Verknüpfungen auf dem Desktop anlegen, sie funktionieren aber nicht.     Sorry, I can't answer in German, but the problem lies in Rox getting the file type wrong. It was wrong on my system too. You need to launch Rox as root, go to /usr/bin, right-click on midori and change its file type to x-executable application.
