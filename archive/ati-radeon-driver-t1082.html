    If you have a pretty much stock system you could use the mepis tools.

    control-center>>hardware>>configure x server

    This will download the driver and install it for you. Check your /etc/apt/sources.list to make sure the mepis repo is still enabled. I think anti disables it by default.

    Right click and select run. Type gksu leafpad /etc/apt/sources.list. Remove the # in front of the mepis repo, save and close then use the mepis x-config as mentioned earlier.
