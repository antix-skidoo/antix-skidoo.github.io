Use the command:

Code: Select all

 setxkbmap -variant intl
To make it permanent, edit the XKBVARIANT line in /etc/default/keyboard:

Code: Select all

XKBVARIANT="intl"
A complete list of all possible variants is in the file /usr/share/X11/xkb/rules/xorg.lst
