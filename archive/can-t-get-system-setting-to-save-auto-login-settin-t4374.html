I Googled for the error message and got some hits. This seems to be a KDE bug. One suggested workaround was to use:

Code: Select all

$ kdesudo systemsettings
If that doesn't work, I suggest you use Google to find other solutions. Most people here don't use KDE and since this is a KDE specific problem, you will probably have better luck with Google.
