Getting ready for Final release!

32 bit version now available and the 64 bit version will be uploaded shortly.

New to RC1:
--Many bug fixes and improvements
--Lots of additions to MX Package Installer
--Latest stable Firefox and Thunderbird
--Updated full Users Manual

64 bit version now ships with the 4.2 Debian Backports kernel since some testers reported issues with the 4.2.3 liquorix kernel. Interested to hear how it performs on individual setups.
