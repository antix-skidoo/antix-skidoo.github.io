1. yep, and libdvdcss is available via simple install in our metapackage-installer. Most codecs work OOTB.
2. Don't know, but all the apps you mention are available in the debian repos, which are enabled by default.
3. Locale stuff is installed by default, but you'll need to configure. Language choice is available on the live media via an Fkey menu, and those settings will carry over to install. You might need to mess with the keyboard setting though, not sure. In MX edition, the XFCE Keyboard app will let you add the colemak layout to you standard keyboard.
4. antiX-15 is more up to date than Crunchbang 11, since now based on Jessie. Lots of themes available, including some nice dark ones, for icewm, jwm, and fluxbox. I don't think it will cost you any ram, its just themes, and the light desktop is just another theme. the MX edition of antiX is still debian-wheezy based, so basically the same as #!-11, but with a community repo providing backported packages. I've got no complaints...
5. I haven't tried Linux Lite in a while, but both antiX and MX are likely lighter on ram usage, as Linux Lite uses XFCE for a desktop and pulseaudio for the sound setup. antiX uses a plainer WM setup and goes with straight alsa for audio. I would guess antiX would be at least 60 mb lighter than Linux Lite, but that's a guess. MX edition uses about 150 mb on startup (depending on enabled services) and still uses the XFCE desktop, but not pulseaudio by default.
6. I use around 500 mb with a modern chrome browser and 3 tabs. Otherwise, it depends on the apps. XMMS or moc for music don't take up hardly anything.
7. My core I5 system takes around 20 seconds to boot to antiX-15 64bit. That is on a spinning hard drive, non-SSD. boot time for MX edition is slightly longer, probably around 30 seconds. This will be dependent on your particular hardware.
8. antiX -15 proper does not use thunar/xfce. But, file searching is available via the searchmonkey app and through the spacefm file manager search utility, which I happen to like. If you are talking about antiX MX edition, which does feature the XFCE/Thunar setup, then most of what I've said above still applies, and an app called Catfish is available for file searching. Thunar doesn't support file searching itself, but the MX team did integrate file searching into thunar via right click menu custom actions.

If you post your hardware specs, we might be able to offer some suggestion about which edition you should install. But if your computer was built in the last 5 years or so, MX edition should really fly on it. There is a dedicated MX community as well at forum.mxlinux.org and a website at mxlinux.org.

The live media for both antiX-proper and the MX edition are quite good. Give them a try and see if you like it. I run one of my systems exclusively from a liveUSB stick, never installed to the hard drive. Once its booted up, its still quite fast.

Hope all that helps.

You are in luck! Setting the keyboard varient in antiX live is very easy. Once you have the live system configured the way you want, the settings should carry over when you install.
