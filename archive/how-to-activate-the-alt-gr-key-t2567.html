Try
dpkg-reconfigure keyboard-configuration

Or manually edit /etc/defaults/keyboard to this:

Code: Select all


XKBMODEL="pc105"
XKBLAYOUT="es,us"
XKBVARIANT=""
XKBOPTIONS="lv3:ralt_switch,compose:lwin,grp:alt_shift_toggle"
I'm presuming you have that file
