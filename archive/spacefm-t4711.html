Yes it can you will need to enable it in etc/udevil.conf and install the ftp mount package. I cannot remember the name of the package but a search in synaptic or Google should show it. Another option is instal the ftp mount package and add an entry into etc/fstab and use the mount command manually or make a bookmark / custom action in spacefm. The majority of setting this up should be outlined in D.O. 's samba share videos. Just change anything that references samba to ftp. This is actually how I handle website configuration when I am in the lan of the server.... that or using nfs instead of ftp.
