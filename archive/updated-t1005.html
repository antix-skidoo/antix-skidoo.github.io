fluxbox dosen't have start bar icewm has.
fluxbox dosen't add items to it's menu by it self you have to add by hand: right click > settings > control centre > desktop > edit fluxbox settings > menu then add a new line where you want to be in the menu :
[exec] (name you want shown in menu) {program name like in terminal}
for example:
[exec] (Firefox 3) {firefox}

Fluxbox is quite different than most WM. It does not have the ability to add apps to the menu automagically, you must add them by hand. This is not difficult.

If you open /home/username/.fluxbox/menu in leafpad you will see the syntax used. It's very simple really and easy to customize. The first thing I would do is make a back up copy just in case you error. Then go ahead and edit the file. Just copy the syntax.

[exec] (name) {command}
[exec] (web browser) {iceweasel}

Just be careful when adding sub-menus.
