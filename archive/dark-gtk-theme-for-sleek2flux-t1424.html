Attached is a dark theme I made that will go very nicely with sleek2flux. It is based from xfce, so you will want to

Code: Select all

apt-get install gtk2-engines-xfce
This engine is lightweight, and based from the gnome engine already installed.

The name of the theme is 'darkcyan'. you can place it in ~/.themes, or make the folder .themes in your 'home' directory if it does not exist. To use in root applications, the 'deepflux' folder will need to be copied to /usr/share/themes as root.

Mousepad is a simple text editor I like to use because it displays the system gtk colors, and deepcyan was made for readability/easyontheeyes:) If you

Code: Select all

 apt-get install mousepad
, you will also drag in the xfce4-mcs-manager, and the xfce4-mcs-plugins, which will allow you to change your settings for fonts, icons, antialiasing, hinting, and themes using the light weight xfce 'user interface' This can be found through the menu by going to > applications > system > administration > xfce-settings-show and then clicking on 'user interface'

To have the xfce user interface save and set your settings each time you log into fluxbox, just add the line :

Code: Select all

xfce-mcs-manager &
to your ~/.fluxbox/startup file BEFORE the line that says:

Code: Select all

exec /usr/bin/fluxbox
