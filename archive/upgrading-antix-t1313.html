You can upgrade the apps by using apt-get or synaptic (or smxi script).
I would do this regularly to save on huge downloads and also as there is less chance of upgrades breaking.

You should also change the /etc/apt/sources to include this line:

#Mepis8
deb
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"ftp://ftp.mepis.com/mepis/"
linktext was:"ftp://ftp.mepis.com/mepis/"
====================================
" onclick="window.open(this.href);return false mepis-8.0 main

If you wish, you can upgrade the kernel. To get the latest MEPIS kernel see this post.

sidux and Debian kernels work in antiX too. Best to use the smxi script.
