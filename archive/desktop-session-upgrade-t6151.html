You should answer Y (yes) to accepting the maintainers version.
If you choose no, or have already chosen no, then you might need to edit a file to stop a yad dialog at desktop session from not closing.

If so do this:

Code: Select all

Disable it in control centre/session/global-desktop-session/desktop-session.conf

Change line 37"Notification Dialog". dialog to read false

You might need to logout and in again
Do the same if you have a desktop-session.conf in your home/username/.desktop-session directory.

ie control centre/session/user desktop session/desktop-session.conf
