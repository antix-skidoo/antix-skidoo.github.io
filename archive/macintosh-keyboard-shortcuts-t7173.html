For changing the keyboard layout, the simplest thing to do is to edit the file /etc/default/keyboard and then log out and log in again to restart X.  All of the options available to you are in the file /usr/share/X11/xkb/rules/xorg.lst.  I suggest you start with using:

Code: Select all

XKBMODEL="macbook78"
