I am making the title simple so when somebody googles skype and AntiX it will bring them to this post. I am doing this on a IBM A22m running one of the AntiX 8.5rcs

So you know what my sound driver and Alsa Mixer Version is on this laptop.

I downloaded and install the Debian Lenny pkg from Skypes site. Downloaded to Download Folder in /home/harry (if you don't have a Download folder in home, make one)

Manuver to Download folder in PCmanFM and right click on empty space and select open Terminal.

UPDATE

Since PCManFM Update. Right click open Terminal not available any more. Thanks Secipolla;
Actually there's no right-click option to open a terminal in pcmanfm anymore. To open a terminal in the same location that the file manager (pcmanfm) is opened, just hit F4 or go to Tools>Open the current folder in terminal (F4)
If Default terminal isn't set yet. Select Rox Terminal from drop down list and OK it. Right click again and select terminal.
Become root in terminal by typing sux and enter your password.
as root

Code: Select all

dpkg -i skype-debian_2.1.0.81_1_i386.deb
Hit enter. Wait for it to finish installing. I log in and out so as to refresh menu list.

Now don't open skype yet. I wanted to make sure my New External Mic would work first. My laptop internal mic either does not work or there isn't one. Not sure.

mhWaveedit which is in Start>Applications>Multimedia uses xmixer as the mixer in Antix. xmixer does not exist. So to fix this. Open pcmanfm or rox. View hidden files. Go to .mhwaveedit file and open it. Inside is config. Left double click to open as text and edit the line mixerUtility = xmixer to mixerUtility = gamix.
Log in and out is what I did to make sure it resets.
Open mhwaveedit and go to play in taskbar and pick record.
When record opens select open mixer. Gamix should now open up.

Note: if you get a Alsa Mixer is running or not supported format or whatever when you open up record. Just reboot to start over from scratch after editing .mhwaveedit/config.
Below I am showing how my setup now looks.


Start recording. If the needles are jumping you are good to go. If not you need to configure alsamixer (open terminal and type in alsamixer and adjust as necessary) so at least it looks similar to mine. By the way. Gamix is the GUI frontend for alsamixer so you can also use it instead of typing alsamixer in terminal. Use whichever feels more comfortable for you.
