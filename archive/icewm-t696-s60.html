Hey, cowboy. If you want to change the pcmanfm folder icon, do this:

1. Find the image you want to use.
2. Copy it to the KorstroDark theme folder:
cp /path/folder_icon.ext ~/.icewm/themes/KorstroDark/icons/folder_16x16.xpm
3. Restart icewm using the"start" menu.

You should now see the icon of your choice. Of course you may want to back up the folder icon first. I think this switch will do it for you, but I'm only 95% sure:
cp -b /path/folder_icon.ext ~/.icewm/themes/KorstroDark/icons/folder_16x16.xpm
