    • A review of the beta in Spanish:
      ========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
      url was:"http://www.linuxadictos.com/antix-15-distro-muy-ligera-con-el-kernel-de-linux-4-0-y-sysvinit.html"
      linktext was:"antiX 15, very lightweight distro with the Linux kernel 4.0 and SysVinit"
      ====================================
      :
      antiX is a GNU / Linux light and spiky, based on Debian Testing and with a design that has always tried to offer many possibilities despite using a window manager like Fluxbox. An example of this versatility in terms of customization options is being able to use a dark or light theme choose one or metallic, for which we press F6 during startup and select.

      This is only one of the new beta of antiX 15, available from a few hours ago and which also incorporates the Linux kernel 4.0. This offers not only the stability and security of a good new version and support for a lot of hardware newcomer but also the enormous benefit of live patching, which is nothing other than the power to patch without rebooting the team.

      This beta has been created using the Devuan debootstrap of the Debian fork arose among those dissatisfied users Systemd so in this distro will have SysVinit. And as for the design it brings as a novelty, besides the aforementioned metallic theme, a new menu and a menu editor, but also fixed a bug that was with wallpaper changer.

      antiX is available for platforms 32 and 64 bits, and can be installed on computers with only 64 MB of RAM (although in those cases will have to appeal to a swap file). We can get it from the download section on the official website.

      :
      They are not the most popular or the most attractive, but there are flavors of Linux for everything and when it comes back to life old cascarrias, antiX and 4MLinux are two good examples of light distributions. We take the recent launches of both to make a hole.

      Starting by antiX 15, is based on Debian 8 dispenses systemd and includes a custom kernel 4.0.5. As SpaceFM desktop uses a fork PCManFM (the file manager of LXDE) and as a window manager IceWM, but also packaged Fluxbox and JWM.

      antiX 15 should be able to be used on computers from 64 MB RAM and Pentium II processor. You can download the latest version at SourceForge.
    • Another nice mention in Spanish:
      ========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
      url was:"http://www.oneopensource.it/06/07/2015/antix-15-basato-su-debian-8-ma-senza-systemd/"
      linktext was:"antiX 15, based on Debian 8 but without systemd"
      ====================================
      :
      News from Greece, the development team of the popular GNU / Linux antiX recently announced the release of new version of antiX 15.

      After a long development cycle composed of two beta releases and a release candidate has finally arrived to the availability of a final official only in the past few hours. But not only, antiX 15 is based on Debian 8, codenamed"Jessie," and carries with it the kernel Linux 4.0.5.

      The codename of this release is"Killah P" and marks a separation with one of the components most loved / hated the view GNU / Linux: systemd. antiX 15 comes indeed without this system operator and follows a series of other distributions in the same way that they have decided not to adopt it.

      The distribution comes with a large number of OSS software already installed, from the classics to LibreOffice Mozilla Firefox until you get to SpaceFM, ROX-Filer, WinFF, Asunder, Wicd, Ceni, GNOME-PPP, Geany, Leafpad, Unetbootin, antix2usb, iso-snapshot, Grub Customizer, HexChat, LuckyBackup, Simple Scan, Transmission, wingrid-antix, Xfburn, Adobe Flash Player, nano, vim, irssi, MPV and Alphine.

      A distribution not really suitable for all all types of users, but sufficiently ready for use in the professional field.
