As for Dave reminding me, I had posted this tip in the wrong thread and so here it is:
Hi Dave, Linux Mint as we know just released a Debian based version. And as a big community now we get alot of info. Who would guess it would be so easy to install the proprietary ATI driver:
1. Open Synaptic Package Manager and search for: fglrx-control (it will install all necessary fglrx packages) and Apply.

2. Open up a terminal type the following 2 commands.
su
aticonfig --initial

3. ctrl+alt+backspace (restart x)
