For now, you should be able to start it from a terminal window with the command:

Code: Select all

sudo bleachbit
It will ask yo for your user password, not the root password.

You could also add a line in ~/.fluxbox/menu to add bleachbit to your main fluxbox menu:

Code: Select all

[exec] (Bleachbit) {gksu bleachbit}
I think this one will pop open a window asking you for the root password, but I'm not certain.

There will be people here tomorrow that know more about the menus then I do. They might be able to help you fix the actual problem instead of just work around it.
