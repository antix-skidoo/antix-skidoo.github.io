I would like testers to try out/comment on the following.

* isolinux used instead of grub in live mode (see menu)
* hybridiso is now incorporated which means that you can install to a usb stick with one command (live only, no persistence)

dd if=/path-to-iso-file/nameofantiX.iso of=/dev/sdx

where if=input file of=output file

eg dd if=/home/antiX/antiX-M11-686.iso of=/dev/sdc

* new wallpaper

We are, I hope, close to going final. Let us know what you think

edit: added xdelta patch for those that have pt1

edit2: For live test out F2and F3 options.
If you want to use xorg.conf (maybe default resolution is wrong or you prefer another one or testing in Virtualbox) use cheat xres=a x b eg xres=1024x768. You may also want to set the vga. Use vga=xxx ie vga=791
If your language or timezone is not in F2/F3 list, then use cheat lang=xx and tz=x/y eg lang=fr tz=Europe/Athens

F1 shows a list of available cheats.

Othe new features include

* Dave's inxi-gui (with some slight modifications)
* remaster script now includes making a final isohybrid .iso file
* qmmp replaces xmms
* a couple of new fluxbox themes

* anything else - suggestions

* add more options to antix2usb script such as cheats

If annoyed by the initial delay with fluxbox menu opening, do this.

In ~./fluxbox/startup file make this change

#auto-fluxbox-menu.sh &

Now this means that the menu will not auto-update on every login, so you will have to use that feature in the fluxbox menu if you install any apps. Desktop-> Fluxbox Settings-> Auto-update menu then Desktop-> Fluxbox Settings->Reconfigure
