edit / etc/slim.conf and set your user as the default. if you want autologin enable it.

default_user dmk-antiX (to have the username entered on boot)
auto_login yes (to login without requiring a password)
focus_password yes (to have the password field focused just to enter it. i don't know if this works with a single line login like in antix)
