An option that I like very much is wbar and wbarconfig, this is similar to the mac applications bar. google search wbar and wbarconfig and download wbar_1.3.3_i386.deb and wbarconf_0.7.2-1_i386.deb and install them using dpkg -i in the terminal.
EX: dpkg -i wbar_1.3.3_i386.deb
ps. make sure you are in the downloaded folder and are root.
