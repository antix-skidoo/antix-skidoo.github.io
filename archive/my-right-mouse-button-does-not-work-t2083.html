antiX uses icewm and fluxbox window managers so it does not work in the same way as full desktop environments such as gnome, kde or xfce.
To add a shortcut to the icewm toolbar at the bottom, you'll need to edit the toolbar file.
In antiXcontrol centre->Edit icewm_>Toolbar.
Just follow the same syntax.

Alternatively, you could use rox-panel and that does allow drag and drop of an app into the panel.
