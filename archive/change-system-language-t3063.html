For the menus, control center, etc. You will need to copy the files from /usr/share/antiX/localisation/LANGUAGE-WANTED to the appropriate folders, ex menus to ~/.fluxbox, ~/.icewm, and the control center and contol center apps to /usr/local/bin. Iceape and iceweasel you will need to change in their prefferances I believe,as to the other apps I do not know.

I normally run the file manager as root to accomplish this, but if it is a command that you prefer than do
cp -r /usr/share/antiX/localisation/LANGUAGE-WANTED/fluxbox/* /home/USER/.fluxbox
cp -r /usr/share/antiX/localisation/LANGUAGE-WANTED/icewm/* /home/USER/.icewm
cp -r /usr/share/antiX/localisation/LANGUAGE-WANTED/local-bin/* /usr/local/bin

and for future users
cp -r /usr/share/antiX/localisation/LANGUAGE-WANTED/fluxbox/* /etc/skel/.fluxbox
cp -r /usr/share/antiX/localisation/LANGUAGE-WANTED/icewm/* /etc/skel/.icewm
