If mounting pendrive in 12.

I used antixcc (crossed wrenches in icewm toolbar)>Disks>Mount Connected Devices.

If using 13.2. Mine automounts in Spacefm because I only use Icewm with no icons desktop and my icewm startup line reads like so

Code: Select all

# run spacefm -d to automount plugin devices and popup spacefm filer.
# Do NOT enable if you use spacefm desktop.
spacefm -d &
Editing Icewm startup can be done in antixcc>Desktop>Edit Icewm Settings
