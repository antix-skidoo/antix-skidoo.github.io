anti, I have a fix for tidesktool.
I was observing that the mouse hover tip didn't work, then I checked tidesktool and noted that this feature's variable didn't match with idesk's wiki (
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========

====================================
). So I fixed tidesktool and now it works.
You must replace

Code: Select all

CaptionTip
for

Code: Select all

ToolTip.Caption
matching the case and in all occurrences.

antiX 8.5 uses a different menu approach teresaejunior and rokytnji. It doesn't use Debian's menu anymore.
nats, your 'Applications' line should be

Code: Select all

menufile"Applications" /usr/share/icons/gTangish-2.0a1/32x32/apps/blam.png application
in line 25 (I think) of ~/.icewm/menu.
After you add it, run (Super key + Space bar)

Code: Select all

icewm_menu2.sh
