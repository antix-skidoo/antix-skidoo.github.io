I recently installed Google Earth and ran into :

Fatal error in __driConfigOptions line 1, column 0: unknown encoding.
Google Earth has caught signal 6.

In order to find out the error, just type

Code: Select all

googleearth
into a terminal. Any errors will be shown.

The solution I have appears to be okay, and Google Earth works now on my system. That being said, use at your own risk.

Open up a terminal
su to root
then:

Code: Select all

cd /usr/lib/googleearth/
then:

Code: Select all

wget http://librarian.launchpad.net/7037027/libGL.so.1 -O libGL.so.1
After this, googleearth should start just fine. I do notice however, that it takes 10-15 seconds for the earth to initially show up.

Also, If somebody else tries this fix, please post your results.
