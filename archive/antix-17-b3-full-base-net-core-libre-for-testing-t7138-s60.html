koolstofje wrote:17-B3 CPU usage in rest is 7%
16.2 CPU in rest is 4%

Can i change?
Are you running live or installed? You can try to beat this back by using the htop program and then using the <F6> key to sort by PERCENT_CPU. If you don't have any extra programs running then the next place to look is unneeded services being launched at start up. The command to control these services is sysv-rc-conf which needs to be run in a console. It is probably also available in the Control Centre.

If you run htop on both systems you may be able to identify what has changed. OTOH, the difference may be due to changes inside of programs that you (we) don't have control over. Every year, the same programs take up more and more space and use more and more CPU. The growth seems to be keeping up with Moore's Law.
