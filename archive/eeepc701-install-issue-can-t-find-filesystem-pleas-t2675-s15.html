Yes, you should have gotten to a Desktop. Icewm in fact. Now that I know it won't boot up on your Desktop. I feel that the problem is not your 701 or AntiX.

Somehow a step is not working when you made your CD. Either CD is a CDRW that is wore out or a low quality CD or you are burning way too fast maybe. Hard to tell from West Tx.

You are going to have to uninstall U3 software on that Cruzer. At least in my experience I had to, to get the Cruzer to boot a live Iso. The U3 uninstaller is the only successful way I could do this.After removing U3 software. Then I would delete the fat32 partition using Gparted and reformat it as fat32 again to make sure the file system was fresh on the USB drive. Virgin so to speak.

You don't want to format the stick as EXT2 file system. When antix2usb is ran. It will format the drive for you automagically. If you read my thread on how I did it. You will see all the wrong things I did before I finally made a successful live persistent USB of AntiX 8.5.

If you ever get successful making a bootable pendrive and get it to boot on your 701. You can also use the AntiX persistent live usb to install AntiX to your internal 4 gig SSD or just run it with Windows as a dual boot by using the ESC key to switch boot order. This will boot the pendrive seperate from the Windows Drive and any changes made like wallpaper and such will stick on next reboot of flash drive.
