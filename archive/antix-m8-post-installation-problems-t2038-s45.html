For the icon to appear in the toolbar it must be set in ~/.icewm/toolbar . Open /usr/share/icons, choose some folder, see if it has a 16x16 folder (it can be larger but this is the proper size) then choose an icon from it (usually from the 'categories' folder). Then you write in ~/.icewm/toolbar the whole path of the icon replacing /usr/share/icons/gTangish-2.0a1/32x32/categories/preferences-desktop.png.
