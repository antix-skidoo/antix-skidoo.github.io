A Sony Mebius here with antiX on a CF card.

Menu: Logout > Shutdown
and Logout > Reboot
each gives, within a few seconds, an almost black screen
with the fan still running. The system never shuts down
properly.

<Ctrl>+<Alt>+<F1> gives the same almost black screen with
the fan running. No command prompt.

If the power button is immediately pressed and held for
about 5 seconds, the fan stops and the screen becomes
absolutely black.

After booting again,
"ANTIX was not cleanly unmounted, check forced."
After fscking manually and fixing, the the system runs again.

On shutdown, I can wait about ten seconds before holding the
power button. When booting again, the"ANTIX was not cleanly
unmounted" message appears but the automated fsck passes and
manual fsck is not required and soon the system is running
again.

This is not an NTFS or FAT FS problem. The CF has ext2 file
systems. /, /home and swap appear in fstab according to UUID.

Yes, I googled. Nothing really relevant. Ideas?
