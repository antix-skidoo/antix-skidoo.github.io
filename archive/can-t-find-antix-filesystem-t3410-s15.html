A quick Google (Triflex Dual EIDE Controller) led me to the triflex.ko and pata_triflex.ko modules. They are available in recent kernels and there is a pata_triflex.ko module that comes with the most recent antiX testing iso (and probably earlier ones as well). One way to look for it on a system is to run:

Code: Select all

$ find /lib/modules -name"*triflex.ko"
It would be very easy for me to make a new iso that contains this driver in the initrd so it can be used to access your hard drive (and maybe even the cdrom if we are lucky). The hard part will be getting the iso to you because it is too big to email. I will send you an email so we can work out the details.
