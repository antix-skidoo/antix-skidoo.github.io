you can drag .desktop files from /usr/share/applications to the desktop. in the rox-desktop case, links will be made. In the spacefm-desktop case, the desktop file will be copied. Either way will work.

adding to the toolbar requires editing the text file that configures the toolbar. its an easy file to edit. just follow the format for the tool that is already in there



You can also use"edit menus" in the antiX control center to put links in the"Personal Menu", which is handy as well. There is a video about that linked in the personal menu now.
