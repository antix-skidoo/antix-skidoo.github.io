the way I do it (assuming you are using rox pinboard for the desktop, per default).

1. create a link on the desktop by dragging the web browser program file from to the desktop. In my can, I drag iceape from /usr/bin to the desktop.

2. right click on the new link and select edit link.

3. in the arguments field, type in the name of the website you want.

that should do it.
