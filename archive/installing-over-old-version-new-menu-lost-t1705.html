I just installed antiX 8 on my old IBM 600E - I've been using an earlier version for a long time.

I retained my home directory. When I was asked in the installation process if I wanted to use the prior home as my new home I answered"yes." All went well until I booted and discovered that by retaining the old home directory I was also retaining my old fluxbox menu. This menu no longer corresponds to the newly loaded apps, and I cannot find the new menu (the one that was in use when I used the distro live from the cd).

Does the"new" correct menu exist somewhere, and how can I use it instead of my old and obsolete former menu?

The new menu is in /etc/skel/.fluxbox and .icewm

Just copy them over to .fluxbox and .icewm (as user)

You *may* want to copy over the .xinit file too.
