hi I'm Italian and I want to configure my pc keyboard.
I want to set Italian as default language of my keyboard
I've found this:
antiX comes with some keyboard languages pre-set in the antiX control centre.
antiX control centre -> X Window -> Change Keyboard layout
Simply choose the language you wish your keyboard to use.
Note:
The settings are only valid for the session. On logout/reboot the default is (re)set.
The problem is that every time i logout the default is reset and i have to to the same thing every-time i log-in.
How can i overcome this problem?
thanks for your help

utonto,

try this: I don't know the code for Italian, but if it is"it"

In
/etc/default/keyboard (if you have it)
XKBLAYOUT="it"

or in
/etc/default/console-setup
XKBLAYOUT="it"

or in
/etc/X11/xorg.conf
XKBLAYOUT="it"
