Playing a game of"Space Evaders" in a transparent, borderless aterm over a background image of Tokyo. The game is available on recent antiX Live systems via the command"/live/bin/space-evaders". The game is highly configurable. Use --help or -h option for a description of all the options. The game in the picture was configured with:

Code: Select all

/live/bin/space-evaders -r8a10s4e5w100
