Using antiX-16-Live as shipped to obtain clean environment.

Imediately after boot-up disable ad-server blocking to remove it as potential source of problem.

Code: Select all

cat / etc/hosts
127.0.0.1    localhost
127.0.0.1       antix1

# The following lines are desirable for IPv6 capable hosts
::1        localhost ip6-localhost ip6-loopback
fe00::0        ip6-localnet
ff00::0        ip6-mcastprefix
ff02::1        ip6-allnodes
ff02::2        ip6-allrouters
Use Screenshot tool to create identical images in different format

Code: Select all

ls -1lh screenshot*
-rw-r--r-- 1 demo demo 23K Dec 12 05:55 screenshot.jpg
-rw-r--r-- 1 demo demo 37K Dec 12 05:56 screenshot.png
Start Firefox

When trying to upload either screenshot to the forum in a new post it shows the following error message in red
forum error message wrote:Could not upload attachment to /cg_sites/freeforums.org/subdomains/communities/user_data/118000/antix/attachments/1732_1c19312657048204473563e057aa4c47.
Created a zip file containing both screenshots and tried to upload it, but got same error condition.
