The way I keep iceweasel up to date, automatically, is to use this repo:

Code: Select all

deb http://mozilla.debian.net/ wheezy-backports iceweasel-release
I put it in / etc/apt/sources.list.d/various.list.

Run an"apt-get update" and then"apt-get install iceweasel" and it should update to the latest version of the browser without updating the entire distro.

If you want to stick to that firefox tar.bz2 that you downloaded, just unpack the tar.bz2 into its own folder. You will then run firefox from that folder. You'll need to make your own menu entry if you use this method and want to be able to launch from the menu. I have done this in the past, and when I did I just put firefox in a folder in my home folder (~/.local/applications)
