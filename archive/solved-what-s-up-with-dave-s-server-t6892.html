I fixed mine on AntiX 15 with commenting out Daves stable jessie repo and enabling Daves testing repo. Then only installing

Code: Select all

Start-Date: 2017-03-14  10:15:17
Commandline: apt-get install antix-archive-keyring
Upgrade: antix-archive-keyring:i386 (20015.3.14, 20017.3.14)
End-Date: 2017-03-14  10:15:24
Saying yes to over write old.

I then commented out Daves Testing repo <pound sign in front it> in etc/apt/sources.list.d/antix and then re-enabled the Daves stable repo <removed pound sign> and did

Code: Select all

sudo apt-get update
To bring everything back to stock again.

I am just posting what I did for ANTIX 15 users. Not ANTIX 16 users or MX-16 users. I feel I should mention this fact since this is a ANTIX 16 thread.
