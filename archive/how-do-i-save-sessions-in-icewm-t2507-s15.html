ROX-Filer comes with an app, Zero Install Injector, that enables the installation of other ROX apps. I'm not sure but I think it was removed in antiX. If you don't have an 'Add New programs' entry in the menu then it isn't there. To get it, do an 'apt-get install --reinstall rox-filer' and you'll have it.
The go to the ROX-Session page OU812 posted, open Zero Install Injector and drag the link to install it.
