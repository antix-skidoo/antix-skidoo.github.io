Ok guys I finally got it working with the help of this great guy ---> PaulFertser

So all I had to do was install 2 patchs
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"https://dev.openwrt.org/browser/trunk/package/mac80211/patches/"
linktext was:"https://dev.openwrt.org/browser/trunk/p ... 1/patches/"
====================================


(I downloaded the ones which had *regd*)

Then I went to compat wireless website and downloaded the latest(not stable, the next version)

cd compat-wireless-whatever/drivers/net/wireless/ath/
patch -Np5 -i 403-ath_regd_optional.patch (and the same of the other's)


then I ./scripts/driver-select ath9k

make
sudo make install

and it's done __{{emoticon}}__

PS. The reason I used driver-select is because the nvidia(bumblebee) got messed up after installing compat wireless, but now I've got everything working GREAT.
