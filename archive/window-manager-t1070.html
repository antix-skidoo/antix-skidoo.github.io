    Welcome to antiX, VincePOPPY.

    You need to edit, as user the .xinitrc file in you home users directory. add this somewhere in the middle.
    afterstep)
    exec afterstep
    ;;

    And edit, as root, /etc/slim.conf.
    Just add afterstep in this line:
    sessions icewm,xfce,fvwm-crystal,kde,openbox,fluxbox

    I guess that afterstep has a menu file in a folder in the home user directory which you can edit.
