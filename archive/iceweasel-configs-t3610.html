Assuming you are using iceWM ,
antiXControlCenter>Edit IceWm Settings>menu

Change this line:

Code: Select all

prog"Web Browser" /usr/share/pixmaps/iceape.xpm /usr/bin/iceape
To this:

Code: Select all

prog"Web Browser" /usr/share/pixmaps/iceweasel.xpm /usr/bin/iceweasel
Logout, and log back in and everything should work.
