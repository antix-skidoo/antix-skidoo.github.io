My way using /home/harry/wallpaper



Green> make wallpaper directory/folder
Download your wallpaper there >allow no spaces in wallpaper name, ergo: smoke break should be smoke-break or smoke_break.
Open wallpaper setter in antixcc
Red> Select Picture
Drag wallpaper directory to l/h pane in wallpaper changer for easy access later
Purple> pick file (make sure no spaces are in file name or it won't work)
Tourquiose> Hit Open
Blue > Hit Apply

Done.
