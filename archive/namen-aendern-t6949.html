    usermod -d /home/neuername -m neuername

    Code: Select all

    chfn -f"neuer_name" hesse1978

    hesse1978 wrote:Vielen Dank! Leider gibt's ne Fehlermeldung:

    Code: Select all

    root@antix1:/home/user# chfn -f user_1 vorname_nachname
chfn: Benutzer »vorname_nachname« ist nicht vorhanden.
    Das war leider gerade falsch herum. So geht's richtig (mit root-Rechten ausführen):

    Code: Select all

    chfn -f"Vorname Nachname" username
    Alternativ kannst Du chfn übrigens auch interaktiv ausführen (ebenfalls als root):

    Code: Select all

    chfn username
    Dann werden nacheinander für alle Felder (also z. B. auch für die Telefonnummer) die gewünschten Werte abgefragt. Soll ein Feld leer bleiben, drückst Du einfach, ohne einen Wert einzugeben, die Return-Taste.
