Presence of the word"testing" in the topic title is unclear to me.
Also wondering how did you"load up clipit"?
Did you uncomment the"clipit &" line within your ~/.icewm startup file then launch a new session?

You've posted multiple topics mentioning"quirky behavior".
As a troubleshooting rule-out step, I'm inclined to ask:
Are you logging into desktop session using the root user account?
(doing so will introduce quirkiness/breakage, unless you create and populate various missing /root -pathed config files)

Another troubleshooting step:
determine whether the problem is specific to iceWM
DesktopMenu --} Desktops --} Other desktops
switch to a different (JWM, or fluxbox) session.
Open roxterm and launch clipit from roxterm's command prompt.
Does clipit launch successfully? If not, are any error messages displayed in the roxterm window?
