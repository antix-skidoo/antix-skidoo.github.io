I do a end around to play games in dosbox in Antix. I download the free games from
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.classicdosgames.com/all.html"
linktext was:"http://www.classicdosgames.com/all.html"
====================================
and
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.dosgamesarchive.com/"
linktext was:"http://www.dosgamesarchive.com/"
====================================
. I pick the freeware games mostly. I make a directory in /home in rox and name it oldgames. Every game before I download it I make a directory in /home/oldgames and give it the name of that game. I extract the game zip in that game folder.

Then set the run action as /usr/bin/dosbox for the .exe file. I do this on my IBM A22m 1000hz, 256MB ram, 700MB /swap, Antix 8.5 wheezy Laptop. Works OK for some games. Some run too fast (mushroom, it's like centipede). Others run slow but run (like clone invader). Like I said. A end run around.



Works for me.
