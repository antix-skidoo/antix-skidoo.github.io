Caro sbispo, o Synaptic se usa para instalar ou remover programas mas não para a atualização do sistema como um todo. Não me pergunte o porquê mas é talvez por ter sido criado com esta finalidade.
Para a atualização do sistema convém utilizar diretamente o apt-get pela linha de comando. Antes de tudo é preciso executar 'apt-get update' como root para que a base de dados dos repositórios seja atualizada.
Em seguida usa-se o comando 'apt-get dist-upgrade'.

Há alguns problemas que vc encontrará na atualização pelo fato de o antiX 8.2 já ter sido lançado há algum tempo.
Antes de atualizar faça a seguinte modificação:

Code: Select all

gksu leafpad /etc/apt/apt.conf
será pedida a senha de root. Altere a linha seguinte de 20... para 40...

Code: Select all

APT::Cache-Limit 40000000;
Isto é porque há muitas atualizações e é preciso aumentar o cache do apt.
Durante o processo você encontrará um erro para a atualização dos pacotes firmware-linux, para resolvê-lo force a instalação dos mesmos com

Code: Select all

dpkg -i --force-overwrite /var/cache/apt/archives/firmware-linux-nonfree*.deb

Code: Select all

apt-get -f install
O mais simples para a manutenção do sistema é usar o comando 'smxi'. Ele por padrão usa 'apt-get dist-upgrade' para atualizar o sistema e também pode ser usado para a instalação de drivers de vídeo (se for o caso), kernels e outras coisas mais. Há uma postagem aqui na seção em português sobre ele.
