Hello.

Are you using a non-US keyboard. If so make sure your passwords/username can still be typed ok.
Once in X edit the slim.conf file (Control centre->Configure Slim) and make it auto-login with your user name like this.

Code: Select all

# default user, leave blank or remove this line
# for avoid pre-loading the username.
default_user        antiX (use your name-obviously)

auto_login          yes
