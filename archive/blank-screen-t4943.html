When grub starts it shows its menu, at this stage press 'e' to edit, then 'e' to edit the kernel line, likely it is highlighted, add

Code: Select all

rootdelay=10
to that line, press return, then press 'b' to boot the system using those changes. If it all works, when the system is up & running, as the root user, or use sudo, open a terminal, then type

Code: Select all

nano /boot/grub/menu.lst
Alter the kernel line as you did to start up & save the file. It should then work OK next time you boot up.
