    Abre el fichero debian.list con tu editor favorito, con permisos de administrador.

    Veras que todas las lineas son comentarios (empiezan con #)
    Tendrás que descomentar el repositorio de stable o el de testing según te interese uno u otro.

    Por ejemplo para usar el repositorio de stable:

    Code: Select all

    # Debian Stable
# Enabled for maximum stability.
 deb http://ftp.es.debian.org/debian/ wheezy main contrib non-free
 deb http://security.debian.org/ wheezy/updates main contrib non-free
# deb-src http://ftp.es.debian.org/debian/ wheezy main contrib non-free
# deb http://ftp.es.debian.org/debian/ wheezy-backports main contrib non-free
    Ten en cuenta que estoy acostumbrado a usar la consola y editar los ficheros a mano, seguramente se puede hacer lo mismo usando synaptic, pero no lo uso.
