If you have another system, you cna burn the CD. then copy the contents to a USB stick.

Alternately, (I assume you are running MS Windows) the WinRAR utility supposedly can extract the contents of an ISO image. Then you can either extract to, copy to, a USB stick.

Otherwise, you could try getting a USB CDROM drive. Assuming your laptop's BIOS supports it (and the BIOS settings allow it), you can boot the CD from the external CD drive.
