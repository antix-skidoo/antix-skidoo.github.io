Hello All

I've installed AntiX base and then installed xorg/slim/xfce4 and I seem to have a Debian squeeze box as the sources.list has the Squeeze repositories enabled and everything else commented out. Everything works nicely and I like the 'mean and clean' bit.

I'd like to know how to display a directory on a remote server in thunar. The remote server is accessible via ssh and sftp, and I can log into the remote shell using ssh from the command line.

I've installed gigolo but am not getting anywhere! Any ideas welcome.

EDIT: The Antix Forum effect strikes again, found the answer within minutes of posting. In Debian Squeeze, installing gigolo does not automatically install the package gvfs-backends. When you do, it all works.
