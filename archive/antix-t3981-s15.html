That is my original .conkyrc file from my 64bit install. It should be ok for your 32bit install also.

Just change the

Code: Select all

TEXT
${color}antiX-base


antix-base to whatever you are running or grab the one from / etc/skel like Anti mentioned in chat and copy over to /home and name it .conkyrcbk. You may have to rename your running conkyrc as .conkyrccurrent before copying over the skel file before renaming or it will overwrite your current configured conkyrc file. You can rename .conkyrccurrent back to .conkyrc when done and restart conky to run what your screenshot shows.

If you use mine. Name the file .conkyrcbk and put it in /home.
