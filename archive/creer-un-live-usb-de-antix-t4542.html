Suggestion: tu auras une réponse plus rapide si tu postes ta question en anglais (même approximatif)...
Je ne sais pas si tu auras plus de succès (clavier non reconnu) en bootant depuis une clé USB
Je confirme qu'on peut booter sur une clé USB sans disque dur (avec DD retiré)
Bien sûr ton bios permet de booter depuis une clé USB ?
hybridiso is now incorporated which means that you can install to a usb stick with one command (live only, no persistence)

dd if=/path-to-iso-file/nameofantiX.iso of=/dev/sdx

where if=input file of=output file

eg dd if=/home/antiX/antiX-M11-686.iso of=/dev/sdc
donc depuis un linux quelconque, copier octet par octet le fichier iso dans le périphérique USB via la commande dd:

Code: Select all

dd if=chemin/vers/mon_fichier_antix_iso_telecharge.iso of=/dev/sdb
si /dev/sdb est ma clé USB
et la clé doit alors être bootable
C'est alors un antiX live, sans persistance. Tu pourras (si tout se passe bien) faire depuis la session live une installation"hard disk" sur un autre périphérique USB
