    I have a bunch of questions:
    1) Did you try doing

    Code: Select all

    ifdown eth1
    before ifup?
    2) Can you plug the adapter into the main board instead of the hub and try it?
    3) What is the output of these commands:

    Code: Select all

    lsusb | grep Amtek
ifconfig eth1
    4) Have you tried ignoring mnetwork's messages and not rebooting? Set it up the way you want (make sure that Manual is checked and Ndiswrapper unchecked on the General tab), then make sure the Wireless tab has the right SSID and encryption info, then Stop and Start network on the General tab, finally go to the Interfaces tab, select eth1, check Start/restart on Apply, and click Apply? Mnetwork's warnings about"it MUST be restarted" are wrong most of the time in my experience...
    5) Finally, PCMCIA wireless cards are dirt cheap and work fine...
