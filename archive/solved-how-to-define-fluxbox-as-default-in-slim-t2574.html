When I start slim without pressing F1, then antix starts with Icewm. I wanna change this to fluxbox as the default WM. I'm a little bit confused, because of the /etc/slim.conf shows fluxbox as the default WM but icewm appears... __{{emoticon}}__

Make the change in ~/.xinitrc

XINITRC_DEFAULT="startfluxbox"
