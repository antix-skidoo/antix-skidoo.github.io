Hello, hello to everybody.

I only started with linux a few months ago, when I tried several distos on my laptop and ended up with fedora. After I sold that laptop I was looking for an operating system that can run from an external Hard drive (I was traveling and needed a lightweight solution). After trying out grml, wich was too complicated for me to start with I found antiX. I had a few little problems at the start (sound, flash-player, network) but I fixed them with the help of google, this forum and the arch-wiki. This is the fourth Computer I am working with now and everything works pretty good. Except a little problem with flash, but I'll start a seperate Topic for that.

I really enjoy learning more about linux. Every problem is an opportunity to learn. I really enjoy how adjustable all the settings are. I think I will never go back to Windows!
