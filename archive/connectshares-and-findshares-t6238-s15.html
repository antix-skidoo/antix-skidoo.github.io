Shay wrote:Now for AntiX-15, would installing Thunar File manager be of help?
Connectshares is not tied to use within a specific file manager. It is designed to work with any file manager or none at all. It can be used by any app without a file manager such as Thunar running at the same time.

In antiX-15-Full SpaceFM is probably the best comparison to Thunar and in the context of working with network shares it is every bit as capable. A while back a comparison and analysis was done to help users choose the method of working with network shares that suited their personal style, maybe it will help you
post33315.html#p33315

Perhaps some community members might give their views, but my direct answer to your question"...would installing Thunar File manager be of help?" is not really. Remember it is your system and antiX enables you to have it your way.
