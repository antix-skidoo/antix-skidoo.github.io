Is that base, core or full?

Inxi -F will give the full information.

To use the meta package installer, one way is to use run in menu, type install-meta and x the root user box. Give your root password and the options are in Office and localisation in language.

Check what you want and press install. make sure you have a reliable internet connection.
Once finished, the terminal asks for you to type something, just ignore and continue.
