Otter Browser is available in AntiX 15 and 16 if you want it. Just enable MX test repo and run a sudo apt-get update

Code: Select all

harry@biker:~
$ apt search otter-browser
Sorting... Done
Full Text Search... Done
otter-browser/mx 0.9.12-0mx150+1 i386
  Web browser controlled by the user, not vice-versa

harry@biker:~
$ inxi -r
Repos:     Active apt sources in file: /etc/apt/sources.list.d/antix.list
           deb http://antix.daveserver.info/jessie/ jessie main nosystemd
           Active apt sources in file: /etc/apt/sources.list.d/debian.list
           deb http://ftp.ca.debian.org/debian/ jessie main contrib non-free
           deb http://security.debian.org/ jessie/updates main contrib non-free
           Active apt sources in file: /etc/apt/sources.list.d/mx.list
           deb http://mxrepo.com/mx/testrepo/ mx15 test
         
harry@biker:~
Just be sure to Comment out/disable MX test repo after installing Otter browser. You don't want to break your install by leaving the MX test repo on when doing a dist-upgrade on AntiX. As always. Do a sudo apt-get update AGAIN after turning off the MX test repo after the Otter browser install. Something that I am now going to do next after this post. Because I am not going to install it. I only am showing this for info.
