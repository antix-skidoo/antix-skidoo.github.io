If there is a problem with the HD then it may come up sooner or later. But regarding the reinstallation, you can do this:
- backup /var/cache/apt/archives if you like so you can copy the files to the re-installed system and have less apps to download when dist-upgrade.
- install choosing to preserve your current home folder (there's a check box for that and it works very well). For it to work you must choose the same username you're using now.
