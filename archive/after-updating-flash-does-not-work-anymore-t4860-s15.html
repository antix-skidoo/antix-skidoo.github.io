You will probably have to search Adobe archives for flashplugin 10.3, and then install that manually. I had to do this on my JUNKBOX to see any videos. Adobe wants everyone (except Linux users) to update to the newest and greatest.

Once you have found the older flashplugin, install it like the installation file says. It's a PITA, but once you get it installed, keep a copy handy (and all your notes on installation) on a usb or cd for later. Any updates sometimes will mess with the flashplugin, and suddenly you are back to square one.

This is the download and installation:

Code: Select all
$ unzip fp_10.3.183.43_archive.zip && cd fp_10.3.r183.43_archive/10_3_r183_43/
$ tar -xvzf flashplayer_10_3r183_43_linux.tar.gz && cd flashplayer_10_3r183_43_linux
$ sudo mkdir -p /opt/flashplugin-nonfree/v10.3.183.43
$ sudo cp libflashplayer.so /opt/flashplugin-nonfree/v10.3.183.43/

Code: Select all
sudo update-alternatives --install /usr/lib/mozilla/plugins/flash-mozilla.so flash-mozilla.so /opt/flashplugin-nonfree/v10.3.183.43/libflashplayer.so 10

Code: Select all
sudo update-alternatives --config flash-mozilla.so
