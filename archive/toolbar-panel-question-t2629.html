You will need to edit the hidden fluxbox/init file in your home user directory.

Edit this line by choosing a suitable number eg change 16 to 32 for it to be twice as thick.

session.screen0.toolbar.height: 16

You'll need to reconfigure fluxbox Menu->Desktop->Fluxbox Settings->Reconfigure
