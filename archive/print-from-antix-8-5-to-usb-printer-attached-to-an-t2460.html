My printer died, so in the interim while I shop for a new printer, I decided to try and print from my desktop antiX 8.5 that is connected via ethernet (I have at&t uverse), to my wifes Imac , it is also connected via ethernet.
First went to the Imac under System Preferences>Internet & Wireless>Sharing, clicked on printer sharing
then opened System Preferences> Internet & Wireless> Network and got the ip address of my wifes computer
Then went to antiX opened browser and go to:


========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://localhost:631"
linktext was:"http://localhost:631"
====================================


to access the cups printing system:
clicked on Administration> Edit Configuration File
added my wifes ip address like so:

BrowsePoll 192.168.x.xxx:631
be sure to put the actual address in!

click on Save Changes,
this will restart the cups server,

If your Imac goes to sleep, you must wake it up before you run the next step, also under Administration:
click on Find New Printers and after a delay you should see the printers available .

YMMV

Hope it helps.
