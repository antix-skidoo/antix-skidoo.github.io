When upgrading to a newer MEPIS kernel you may get an error such as this.

Code: Select all

Errors were encountered while processing:
 /var/cache/apt/archives/firmware-linux-free_2.6.32-10mepis1_all.deb
E: Sub-process /usr/bin/dpkg returned an error code (1)
Solution, in a root terminal:

Code: Select all

dpkg -i --force-overwrite /var/cache/apt/archives/firmware-linux-free_2.6.32-10mepis1_all.deb
apt-get -f install
make sure that it is the correct .deb.
