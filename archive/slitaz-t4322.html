    Before I've replaced Ubuntu by AntiX as my main, I also tested Slitaz. It used less memory, than AntiX ( 30 MB vs. 40.5 MB , default desktop, minimalist config). The main reasons to reject it were: 1. Small repository & could not install debian packages. 2. The get-skype script failed. 3. No localisation.

    We used to chat on skype a few hours a month, saving a lot on phone bills, and I like to have as much packages avalaible, as possible. So I have deleted all non-debian-based distros from the candidate list. If you just need a really lightweight live CD/usb & a browser, then Slitaz is probably the best, in terms of RAM usage.
