The default on antiX-M8 is icewm.

Window managers like icewm and fluxbox usually has no problems when using Testing repos, but some of the apps installed in antiX might, especially now that the Testing repo is going to get new stuff from Unstable such as xorg1.6 which *might* cause some initial problems. Some apps in Testing do break (until a fix is found), but not many and not often. However on 'mission-critical' boxes, you should really use the Stable repos only.

I think eriefisher's suggestion is good. ie use one box as a sort of"Test" box to see what is happening.
