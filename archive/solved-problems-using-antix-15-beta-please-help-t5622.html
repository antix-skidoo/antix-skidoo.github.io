1. You can install the mate-panel. Its in the default repos. You will want to install mate-panel and mate-power-manager, as the power-manager brings the brightness plugin.

2. not sure on this one...

3. you can disable the icewem toolbar by opening Control Centre>Edit Icewm Settings, go to the"preferences" file tab and edit"ShowTaskBar=1" to"ShowTaskBar=0" and restart icewm. The mate-panel can be run with"mate-panel" from the run box on the menu. You can add that to your default startup if you find that you like it. mate-power-manager is run by"mate-power-manager" and will show up in the mate system area. Don't worry, the icewm menu is still accesible via a right-click on the desktop.

I would not leave deb-multimedia enabled all the time. You will break something, almost guaranteed. When I use that repo, I install the app then disable the repo. Its too dangerous to leave it enable as you can break your system during an upgrade.
