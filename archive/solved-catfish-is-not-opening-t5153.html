This is a bug in Debian's catfish.

You could try this.

Edit /usr/bin/catfish (as root)
Change

Code: Select all

%python% /usr/share/catfish/bin/catfish.py"$@"
to

Code: Select all

python /usr/share/catfish/catfish.py"$@"
