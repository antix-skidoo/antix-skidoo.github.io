Pierre, if you want to use Icewm you will have to be patient and configure it. In antiX full it comes already configured.

For Icewm manual, open /usr/share/doc/icewm/html/icewm.html

The parameters for Icewm are in /usr/share/icewm/.
First create a folder in your user folder called .icewm (with a dot '.' in front - it will only show up if you enable hiden objects), then copy keys, menu, preferences, toolbar and winoptions from /usr/share/icewm/ to it.
Then you can tweak Icewm's preferences as you like.

For you to be able to reboot and shutdown first you have to edit .icewm/preferences. Open it with leafpad, press ctrl+f and search for shutdown. Then leave it like this

Code: Select all

#  Command to shutdown the system
ShutdownCommand="sudo halt"

#  Command to reboot the system
RebootCommand="sudo reboot"
and save.

Then you have to edit /etc/sudoers so it won't ask for a password when you reboot or shutdown.
Open a terminal as root or type

Code: Select all

su root
in a normal terminal, then

Code: Select all

visudo
then add this line

Code: Select all

%users ALL=NOPASSWD: /sbin/halt, /sbin/reboot
press ctrl+o to save, confirm and then ctrl+x to exit.

To have the Wicd tray icon do this: create another file inside ~/.icewm and call it startup (so it will be ~/.icewm/startup) with this content

Code: Select all

#!/bin/bash

sleep 2 && wicd-client &
then open rox file manager and go to ~/.icewm, right-click the startup file, select properties and check the 'Exec' boxes in 'User', 'Group' and 'World'.

Then logout and log back in for the changes to take action.

For the Control Center, if you have an antiX full cd, boot into it, go to /usr/local/bin and find antixcc-fr.sh then copy it to /usr/local/bin of your installed antiX. And to make it appear in the toolbar, open ~/.icewm/toolbar and add

Code: Select all

prog"Centre de contrôle" /usr/share/icons/gTangish-2.0a1/32x32/categories/preferences-desktop.png antixcc-fr.sh
