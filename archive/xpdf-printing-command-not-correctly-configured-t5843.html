Just another small inconvenience that might be corrected in Antix 15.1.

The printing command used by Xpdf is set to 'lpr'. When I tried to print, nothing happened and no error message is displayed, even when starting it from a terminal. I quickly found out that I needed just 'lp'. Unfortunately, this setting is not saved, but you can edit the config file / etc/xpdf/xpdfrc (or a local copy ~/.xpdfrc).

By the way, I was very pleased to see that Antix automatically discovered and correctly configured my DeskJet 5550 printer. Sharing it on my home network was very easy too.
