on the 32 bit versions of antiX, installing skype is as easy opening Menu>System Tools>metapackage-installer and selecting"Skype" from the messaging area.

In the 64bit version of antiX, you need to enable multiarch first with this command (as root)

Code: Select all

dpkg --add-architecture i386
before installing skype.
