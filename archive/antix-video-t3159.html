Script for smaller screen on EEEPC 701SD is

Code: Select all

ffmpeg -f x11grab -s 800x480 -r 24 -an -threads 2 -sameq -i :0.0 screencast.avi
Short version 6MB video<Edit: I removed the video>

Long version 46MB video <Edit: I removed the video>


Notice the change on the script from 1024x768 to 800x400. You can use xrandr in terminal to see what you are using for screen size to enter in AntiX.

Code: Select all

$ xrandr
Screen 0: minimum 320 x 200, current 800 x 480, maximum 4096 x 4096
LVDS1 connected 800x480+0+0 (normal left inverted right x axis y axis) 0mm x 0mm
   800x480        60.0*+
   640x480        59.9
I have played around with this small netbook and hooked a 21.5 inch widescreen
external HP LCD monitor and AntiX also auto magically picks it up and gives me correct resolution in it also. I tested the download in my link and it works also.

Command I used AntiX 13 64bit

Code: Select all

ffmpeg -f alsa -ac 2 -i hw:0,0 -f x11grab -s 1366x768 -r 25 -i :0.0 -sameq /home/harry/Video/test2.avi
Done on base iso beta 1 install on my Acer Aspire 5534. Just screwing around to see if the command would work. Works in xdelta RC1 AntiX 13 full iso also.
You will have to change the screen resolution though to fit your screen. Mine is 1366x768. Change to what ever you are using eg: 1024x768.
