techpatterns.com/sgfxi -> smxi.org/sgfxi
techpatterns.com/svmi -> smxi.org/svmi
techpatterns.com/smxi.zip -> smxi.org/smxi.zip

# the techpatterns.com full path urls are changed:
techpatterns.com/downloads/distro/smxi -> smxi.org/sm/smxi
techpatterns.com/downloads/distro/sgfxi -> smxi.org/sg/sgfxi
techpatterns.com/downloads/distro/svmi -> smxi.org/sv/svmi

# and the documentation is also updated:
techpatterns.com/docs/ -> smxi.org/docs/
