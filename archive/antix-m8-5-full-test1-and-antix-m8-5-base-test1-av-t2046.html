The main feature of this test release is to 'remaster on the fly' ie use the live usb with persist option, add/remove apps as you please and then install all that to hard disk. Changes should carry over (Changes in /home/demo might not).
Or run like puppy linux on a stick as live usb and changes will be saved if persist option is chosen.
Or simply install as usual.

Bugs from previous releases should be fixed, so no insserv errors and once installed enabling fast boot should work without error.
(#dpkg-reconfigure sysv-rc)

Live boot should be faster than previous antiX.

Please test out the antiX2usb script and persistence.

New:
desktop=fluxbox cheatcode at boot menu.
antix2usb gui included. (as is the antix2usb-persist script written by BitJam)
gamix replaces aumix.
imagination slideshow added.
Localisation of antiX apps ie phonebook, mount app, mouse app.
sudoku game added.
sysv-rc-conf replaces rcconf.
geeqie replaces gqview.

Post feedback in this thread.

Added: Could users also try changing the dpi setting in /etc/slim.conf from 96 to 84 or even 72 - logout and let me know what you prefer (give specs via inxi -F). I am thinking of changing the dpi to 84, but would like feedback first.
