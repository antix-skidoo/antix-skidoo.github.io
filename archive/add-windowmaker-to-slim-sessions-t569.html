You also have to edit the .xinitrc file too. I have:

case $1 in
icewm)
exec icewm
;;

openbox)
exec openbox-session
;;

fvwm-crystal)
exec fvwm-crystal
;;

lxsession)
exec lxsession
;;

fluxbox)
exec startfluxbox
;;
*)
exec startfluxbox
;;
esac

I don't know for window maker but I guess you add

windowmaker)
exec wmaker
;;
