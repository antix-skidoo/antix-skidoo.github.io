Larger desktop environments take care of this for you as part of the integration. Since Antix uses light weight window managers this type of integration is not there. These types of window managers are stand alone and will gladly accept any application the user chooses. Fluxbox doesn't rely on Abiword or visa versa whereas Konqueror must have kde and kde brings Konqueror(built in integration).

All of the bits and pieces that make up Antix don't necessarily rely on one another which is why you must tell rox(the file manager) to open a .doc in Abiword(processor). Now that it knows what to do it will alway do it unless you tell it otherwise.
