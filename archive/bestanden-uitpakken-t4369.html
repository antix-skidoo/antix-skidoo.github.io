    Flashplugin moet al zijn geïnstalleerd.

    Hier zijn de instructies hoe dan ook:

    antiX-13-full:
    Toepssingen --> Overig --> Synaptic Pakketbeheer

    "Herladen"

    Snelfilter:"flashplugin"

    etc. ...

    antiX-13-base, antiX-13-core:
    "TerminalVenster"

    sudo apt-get update
    apt-cache search flashplugin
    sudo apt-get install flashplugin-nonfree
