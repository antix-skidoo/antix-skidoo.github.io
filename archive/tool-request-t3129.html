name : T50 Sukhoi PAK FA Mixed Packet Injector.

description :The Sukhoi PAK FA T50 Mixed Packet Injector is a packet injection (packet injection) free, created by Brazilian Nelson Brito, capable of DoS and DDoS attacks using the concept of stress testing. With it you can send a very high number of requests for packets, such that the target will not be able to meet all requests or answer them slowly, so the target may fall or slow down.

Currently, the T50 is able to issue the following requests:

More than one million (1,000,000) packets per second of SYN Flood (+50% of the network uplink) on a network 1000BASE-T (Gigabit Ethernet).

More than 120,000 packets per second of SYN Flood (+60% of the network uplink) in a 100BASE-TX (Fast Ethernet).

The T50 can also send requests for packets of the protocols ICMP, IGMP, UDP and TCP sequence with difference of microseconds.
