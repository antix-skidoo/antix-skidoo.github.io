I set up this antix m8 desktop with fluxbox and fbpanel as the main features. Since i'm also using an eee pc 904ha, I'm showing off the webcam with Cheese. Hi everybody.

fbpanel is setup with cpu monitor and battery monitor, as well as a freedesktop menu (useful when you can't find a new install in the fluxbox menu...)

gtk theme is darkilouche and the icon set is tango-materia. I'm using the NYZ fluxbox them for the colors, with a grey background to match the gtk theme.

iceweasel is set up with an extension to shrink the menu bar out of the way for more screen real-estate, important on a 1024x600 screen.
