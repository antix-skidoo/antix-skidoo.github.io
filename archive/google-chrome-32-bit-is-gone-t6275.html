Google Chrome 32 bit is no longer supported by google. they even went so far as to remove it from their servers. So if you have a 32 bit system with chrome installed, be aware there are no updates.

removing / etc/apt/sources.list.d/google-chrome.list will prevent some weird error messages from popping up.

if you are a 64 bit user. your google-chrome.list file needs a slight change.

add [arch=amd64] to the deb entry like this.

Code: Select all

deb [arch=amd64] http://dl.google.com/linux/chrome/deb/ stable main
update your sources and all should be well.
