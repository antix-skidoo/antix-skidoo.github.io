I hope it is OK to post this in the M8 forum. I have installed Antix M8 to a 8GB SD card in my new ASus EEE pc 1000he. It is working very well..... wireless was set up without a hitch. I use Fluxbox, as I like how easy it is to configure. The volume buttons do not work with the"Fn" button, so I added a couple lines in the"keys" file, and just use them with"ALT" button instead. I haven't found a use yet for the integrated camera, but maybe I will try Skype in the future. Bluetooth is untested, too at this point. The trackpad mouse works right off, with the 1, 2, and 3 finger gestures. (The funky"2-finger zoom" feature doesn't seem to work, but I probably wouldn't use that any way.)
In a word, Antix is THE choice right now for Linux on this box. I tried several others, with less success: Sidux, PClinuxOS, and a couple I have forgotten now.
Did I mention Antix runs like a SCALDED DOG on this thing?????!!!!!!!!!

I thought this would help some users of this new EEEpc model searching for a suitable Linux distro. Nice thing is, it doesn't touch the hard drive, or the Windoze install.....

Thanks again, Anti!!
