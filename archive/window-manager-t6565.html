If I remember correctly, back in 2006 when anticapitalista first released antiX as a supported variation of the MEPIS project, Fluxbox, at that time, was the only window manager included by default. The software has been easy to configure and modify for over a decade now, and very early on, forum participants and contributors discussed the packages to add or include in the various developments and releases. As we made an attempt to make the default system slightly easier, I suggested that we add IceWM as the default window manager. Of course, those discussions led to experimentation and evaluation - ultimately the addition, of several window manager alternatives, and multiple distribution packaging alternatives.

Ironically enough, one of the desktop environments that I'd frequently add to my MEPIS and antiX configurations was Xfce. Today, antiX offers a variation called MX (currently MX-15) that uses Xfce to provide a distribution that is a nice compromise between efficiency and simplicity, features, and performance.

Needless to say, antiX in any form is a good starting point to build the system that you like.
