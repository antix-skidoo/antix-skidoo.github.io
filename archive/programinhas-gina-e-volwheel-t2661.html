====================================
é um bloquinho de notas virtual para a área de trabalho. Taqui um arquivo de instalação .deb pra ele em português (Brasil), com um arquivo desktop (lançador) que pode ser posto em /usr/local/share/applications, /usr/share/applications ou mesmo diretamente na área de trabalho e um ícone pra se colocar em /usr/share/icons/hicolor/128x128/apps/
Pra quem não sabe, instala-se como root e executando

Code: Select all

dpkg -i gina_1.0.0-1_i386.deb
Depois de instalado pode-se obter mais informações com o comando

Code: Select all

man gina

====================================
é um aplicativo de mudança do volume do som com a roda do mouse, um ícone na bandeja.
Eu traduzi ele pro português (Brasil). Pra instalá-lo, basta descomprimir o arquivo, entrar na pasta, abrir um terminal lá, autenticar-se como root e executar

Code: Select all

./install.pl
Leia o README.
O antiX já vem com o AbsVolume no IceWM mas o VolWheel pode ser usado no Fluxbox, no Openbox/LXDE ou mesmo substituir o AbsVolume no IceWM. Basta configurar o seu ambiente de trabalho favorito para iniciá-lo automaticamente.

Eu baixei o arquivo e não estava instalando. Pra poder instalá-lo foi preciso criar o diretório /usr/local/lib/volwheel antes de executar o install.pl
Portanto os comandos foram (com o terminal de root)

Code: Select all

mkdir /usr/local/lib/volwheel && ./install.pl
As dependências dele são libperl5.10, libgtk2-perl e alsa-utils (creio que todas estejam presentes no antiX).
