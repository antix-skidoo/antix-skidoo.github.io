Firstly delete /etc/apt/apt.conf file and then edit the /etc/apt/preferences file to only have this:

Code: Select all

Package: linux-kbuild-2.6.32
Pin: release n=mepis
Pin-Priority: 995
Then apt-get update & apt-get upgrade

There are 2 mplayer packages. One from Debian Testing and one from Debian Multimedia Testing and sometimes they conflict with existing libraries if one of the repos is either disabled or set higher than the other as was the case in /etc/apt/preferences and /etc/apt/apt.conf
