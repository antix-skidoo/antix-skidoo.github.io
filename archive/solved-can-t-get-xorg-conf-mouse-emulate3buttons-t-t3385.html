I found this in an archlinux.org forum. I tried in VirtualBox with the lastest testing version of antiX and it worked. Add the following to /etc/X11/xorg.conf:

Code: Select all

Section"InputClass"
    Identifier"whatever"
    MatchIsPointer"on"
    Option"Emulate3Buttons""on"
EndSection
