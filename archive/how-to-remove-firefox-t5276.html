The deb from Firefox usually installs to /opt, and it may not show up in the PATH. Check first if you see a firefox folder in /opt. Then try adding /opt to PATH, then run your commands again and see if they work.

Code: Select all

PATH=$PATH:/opt
export PATH
This addition is only valid for the current session, would need to be added to ~/.bashrc to make it permanent.
