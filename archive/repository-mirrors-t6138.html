You may wish to change the location of the antiX (MX) repository to one closer to where you are.
At the moment, you have to do this manually.

antiX-15 - edit etc/apt/sources.list.d/antix.list

Code: Select all

#Ecuador, South America
  deb http://mirror.cedia.org.ec/mx-workspace/antix/jessie jessie main nosystemd

Code: Select all

  #Crete, Greece, Europe
  deb http://ftp.cc.uoc.gr/mirrors/linux/mx/antix/jessie jessie main nosystemd
MX-15 - edit etc/apt/sources.list.d/antix.list

Code: Select all

 #Ecuador, South America
  deb http://mirror.cedia.org.ec/mx-workspace/antix/jessie jessie main

Code: Select all

 #Crete, Greece, Europe
  deb http://ftp.cc.uoc.gr/mirrors/linux/mx/antix/jessie jessie main
MX-15 - edit etc/apt/sources.list.d/mx.list

Code: Select all

 #Ecuador, South America
  deb http://mirror.cedia.org.ec/mx-workspace/mx/repo/ mx15 main non-free

Code: Select all

 # Crete, Greece, Europe 
  deb http://ftp.cc.uoc.gr/mirrors/linux/mx/mx/repo/ mx15 main non-free
