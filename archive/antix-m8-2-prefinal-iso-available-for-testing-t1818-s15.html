As I was browsing the icewm preferences file for my icewm config app, I noticed these lines:

Code: Select all

#  Disable Imlib's image/pixmap caches
# DisableImlibCaches=1 # 0/1
And of course I started thinking about the recent problems debian testing users have had updating icewm. So if this is you - remember I'm not using testing - try updating icewm in testing and edit this line in preferences (around line 316) as follows:

Code: Select all

#  Disable Imlib's image/pixmap caches
DisableImlibCaches=0 # 0/1
And then restart icewm using the option in the logout menu.

It couldn't hurt to try.
