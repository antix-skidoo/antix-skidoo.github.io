Yes, users should be careful when upgrading.

I had to add this to my /etc/X11/xorg.conf

Code: Select all

Section"ServerFlags"
        Option    "ignoreABI""true"
EndSection
This is my inxi -G

Code: Select all

$ inxi -G

           GLX Renderer: GeForce 8400 GS/PCI/SSE2/3DNOW! GLX Version: 3.3.0 NVIDIA 285.03
