    kabon wrote:Am assuming that its recommended to update apps for a number of reasons - including security. Is that right? Any security issues with leaving everything as originally installed?
    Depends who you talk to - some sites post security warnings and point you to updated packages. Some sites are in denial. In general, I update my system once a week.
    kabon wrote:If updating is recommended and smxi is the preferred way to update in antiX, will a dist update (without kernel update) disable the control center or break the menu or installed apps like virtualbox?
    The control center should not break during any upgrade as it is a simple script rather than a .deb package. The menus should not break since there is a copy of it stored in the user's home directory. So updates will not affect it. (The updates will affect the files in the root directory.)

    For everything else, someone else will need to jump in.
