    Hello,

    Having installed the brand new antiX M12 on an oldie (PIII Coppermine with 192 MB ram) I thought I could make use of zram. It belongs to the package"antiX-goodies" as I could see, zram is configured as module in the kernel, so I start it with the command:

    Code: Select all

    # /usr/local/bin/zram start
    and I get a message about what to configure in the kernel config file __{{emoticon}}__
