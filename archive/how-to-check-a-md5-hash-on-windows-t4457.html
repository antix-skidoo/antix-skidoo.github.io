You can do it at the command line. Just type

Code: Select all

$ cksum --help
in a terminal, which gives you:
Usage: cksum [FILE]...
or: cksum [OPTION]
Print CRC checksum and byte counts of each FILE.

--help display this help and exit
--version output version information and exit
