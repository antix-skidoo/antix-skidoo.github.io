johnj wrote:Hi All, I have a question about setting key bindings in the JWM desktop.

My question is: Do I set the key bindings for JWM in its jwmrc and what is the letter for the Windows key, is it S?
you edit the ~/.jwm/keys file.

There is an existing key combo defined in the keys file for the root menu (what jwm calls the menu). its Alt+F1.

on my system, in jwm, the Windows/Super key is Mod4, which would be just"4" in the jwm key bindings. as far as I can tell, you have to have another key in combination, like Super+r, to get the root menu. My working entry looks like this:

Code: Select all

<Key mask="4" key="r">root:1</Key>
remember, you have to restart jwm before any change in configuration takes place.
