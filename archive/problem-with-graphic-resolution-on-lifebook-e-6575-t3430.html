Schnake wrote:[...] buildxconfig again, now bash tell me that the command is not knowing and the xorg.conf is still empty....
The buildxconfig command should be at /usr/sbin/buildxconfig. If you are in a terminal as root then /usr/sbin should be part of your path. You seem to be sure you are root so you could try giving the absolute path to the command:

Code: Select all

# /usr/sbin/buildxconfig

Okay i Fixed it now...

I put this Command on the Shell:

Code: Select all

sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade
And restarted, seems like on this command the true Graphic driver was installed and they put my real Resolution and Depth in, so now i can use it!
I used the command for a Problem with Wine wich i fixed too! __{{emoticon}}__
