Bonjour,

* Démarrez Antix Control Center
* Allez vers la page 'System', appuyez le bouton 'Configure system'. Nouvelle dialogue qui s'ouvre.

Sur la page 'Locale':
* Cochez 'Enable locale options' (en haut)
* Cochez 'Enable locale options' (à droite)
* Cochez 'Enable language select'

La reste, vous la trouverez sans doute. Attention: ne JAMAIS appuyer le bouton [OK] sans avoir choisi l'une ou l'autre des options offerts. Vous risquez des degats severes a la système installée. Si vous ne voulez aucune des options offerts, appuyez le bouton [CANCEL] jusqu'a le point ou vous avez fermé tous les dialogues.

Je ne sais pas si la traduisation en francais est vraiment tres complete, mais on peux toujours espérer.

Bonne chance.

Antix CC est fourni par défaut avec antiX, pas avec antiX MX (qui a un Cc similaire celà dit).
Pour l'installer (de mémoire),
1. s'il est dispo :

Code: Select all

apt-cache search antixcc
2. installe le:

Code: Select all

apt install antixcc
