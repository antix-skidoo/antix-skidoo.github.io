Notorik,

Welcome to the antiX forums, I have an hp c6180 connected wirelessly using antiX, the simplest way is to use the metapackage installer, in antiX 8.5 RC2:

go to:

menu>applications>Accessories>metapackage installer> (password)>Office>Hp_printing> Install

thats it, the metapackage will do the rest and you should have a functional install.

Then run in terminal as user

Code: Select all

hp-setup
and setup your printer as you like.


cheers,
oldhoghead

Code: Select all

inxi -F

in antiX 8.0 it's: menu>system>tools>metainstaller>(your password or root for livecd)>office>Hp_printing>install
