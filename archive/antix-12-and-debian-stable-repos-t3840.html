Background
I have a project in which reliability of use (from an end-user perspective) is in the highest rank of priorities. Newer versions of applications are of lesser importance. These point towards using Debian v6 Stable repo rather than Testing.

antiX-12 only lists the Testing or Frozen-Testing (forthcoming v7 Stable) Debian repos, unlike antiX-M11 which also lists an option to use the current v6 Stable repos.

Question 1
Is a fresh installation antiX-12 suitable for immediately re-pointing to exclusively use the current Debian v6 Stable plus antiX repos? This will not be an upgrade from antiX-M11 and will not transfer $HOME of any existing accounts.


The Debian wiki provides guidance on upgrading stable v5 to v6 and will presumably eventually produce similar for v6 to v7.
