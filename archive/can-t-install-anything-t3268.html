Code: Select all

http://ftp.us.debian.org
a while back because their servers have been flaky for me at times. I figured that US traffic (lots of debian testing users) was bogging down the servers so I opted to change to hopefully a less well traveled repository server.

I changed all my

Code: Select all

http://ftp.us.debian.org
to

Code: Select all

 http://ftp.ca.debian.org
which changed my download servers from USA to Canada for apt.

It has been a improvement for me and I live next to the Mexican border.
