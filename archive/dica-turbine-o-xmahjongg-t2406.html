O antiX vem com o xmahjongg (Menu > Aplicativos > Jogos > xmahjongg) que, ao contrário do que pode parecer à primeira vista, tem várias opções extras. O problema é que pra abri-lo com um arranjo diferente ou com pedras diferentes é preciso usar a linha de comando.

Code: Select all

xmahjongg --list
lista as opções de pedras e arranjos. Experimente, por exemplo,

Code: Select all

xmahjongg --layout snake --tileset small
Existe uma interface gráfica pra ele chamada
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.tikmark.com/Mohave/mohave.html"
linktext was:"Mohave"
====================================
. Ela vem com uma porção de novos conjuntos de pedras e temas de fundo, mas a principal comodidade é poder escolher rapidamente e visualmente.
Pois ei-la aqui traduzida ao português, corrigida para englobar todos os 24 arranjos disponíveis e suportar também uns conjuntos do xmahjongg original.
Primeiro descarregue os
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.tikmark.com/Mohave/tilesets.57.tar.gz"
linktext was:"conjuntos de pedras"
====================================
e descomprima-os em /usr/share/games/xmahjongg/tiles/. Também descarregue os
========= SCRAPER REMOVED AN EMBEDDED LINK HERE ===========
url was:"http://www.tikmark.com/Mohave/backgrounds.tar.gz"
linktext was:"fundos"
====================================
e descomprima-os em /usr/share/games/xmahjongg/backgrounds/.
Aí crie umas ligações simbólicas pro mohave enxergar dois conjuntos de pedras do xmahjongg e o fundo original:

Code: Select all

# cd /usr/share/games/xmahjongg/backgrounds/
# ln -s default.gif bkgrd23.gif
# cd ../tiles/
# ln -s gnome.gif mohaveset58.gif && ln -s gnome2.gif mohaveset59.gif
Então descomprima mohave.zip (anexo) em /usr/local/bin/.
Para ter certeza de que ele tem permissão para ser executado por usuários comuns,

Code: Select all

# chmod 755 /usr/local/bin/mohave
Aí é só criar um arquivo .desktop pro mohave pra que ele apareça no menu. Use Menu > Ferramentas > Sistema > Criar arquivos .desktop. Lembre-se de deixá-lo na categoria Games.
