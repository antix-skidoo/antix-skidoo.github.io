Which window manager are you running?
If you're running icewm , the taskbar height can be changed in a couple ways.
The easy way if you like it higher for all themes is to .......
Go to /home/you/.icewm/ and see if you have a file named prefoverride .
If not make one , here's mine.

# Font Specification
TitleFontNameXft = sans-serif:size=12:bold
MenuFontNameXft = sans-serif:size=12
MinimizedWindowFontNameXft = sans-serif:size=12
ActiveButtonFontNameXft = sans-serif:size=12
NormalButtonFontNameXft = sans-serif:size=12
ToolButtonFontNameXft = sans-serif:size=12
NormalWorkspaceFontNameXft = sans-serif:size=12
ActiveWorkspaceFontNameXft = sans-serif:size=12
QuickSwitchFontNameXft = sans-serif:size=12
ListBoxFontNameXft = sans-serif:size=12
StatusFontNameXft = sans-serif:size=12
ToolTipFontNameXft = sans-serif:size=12
ActiveTaskBarFontNameXft = sans-serif:size=12
NormalTaskBarFontNameXft = sans-serif:size=14
ClockFontNameXft = sans-serif:size=16
InputFontNameXft = sans-serif:size=12

MenuIconSize=28
SmallIconSize=28
LargeIconSize=32
HugeIconSize=48

The Menu Icon and or Small icon size will determine the overall height of the taskbar.
This will make the taskbar at least 28 pixel tall.
Experiment , you won't break anything , if you hate it when you're done , just change the settings or delete prefoverride.
The other way is to make a theme and change the button height for the theme ( linux.xpm) in the theme you are running.
You may delete the fonts in this file or comment them out , it will ignore them then.
