Para instalar o smxi, abra um terminal como superusuário (root) e entre

Code: Select all

cd /usr/local/bin;wget smxi.org/smxi.zip;unzip smxi.zip;smxi
Tanto o smxi quanto o sgfxi só devem ser usados fora do ambiente X (gráfico), portanto quando for usá-los inicie/reinicie o sistema, quando aparecer o menu do GRUB destaque a linha do antiX e tecle e, vá até o final da linha do kernel e adicione 3

Code: Select all

/boot/vmlinuz-2.6.27-1-mepis-smp root=/dev/sda4 nomce quiet nosplash vga=791 3
e por fim tecle b para iniciar (ctrl+x se estiver usando o GRUB2). O sistema irá iniciar até o nível 3 (texto), então autentique-se como root.

Os scripts são auto explicativos, para uma visão geral use

Code: Select all

smxi -h
ou

Code: Select all

sgfxi -h
